Основная задача модели это описывать логику и структуру используемых сущностей


**Толстые модели** Rich model
 Модели, которые хранят сущность и определяют поведение сущности



**Thin model** анемичные модели
 Модели, которые хранят только данные сущность



Модели бывают DTO, те данные которые хранятся где-то ещё и вспомогательные

Модели хранятся в Models 
Есть ViewModels, которые хранят данные для передачи в представление
Они хранятся в отдельной папке ViewModels и в названии имеют приставку ViewModel
