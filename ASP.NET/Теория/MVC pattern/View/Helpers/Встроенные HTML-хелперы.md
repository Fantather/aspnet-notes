В ASP.NET Core MVC встроенные HTML-хелперы представляют собой методы расширения, доступные во View через свойство `@Html`. Они генерируют стандартные HTML-теги и автоматически привязывают их к данным модели или маршрутизации


## Типы встроенных хелперов
**Слабо типизированные хелперы**
  Принимают имя свойства в виде *строки* (например, `@Html.TextBox("Name")`). Ошибки в имени обнаруживаются только во время выполнения программы

**Строго типизированные хелперы**
  Заканчиваются на суффикс `For` (например, `@Html.TextBoxFor(m => m.Name)`). Используют лямбда-выражения для работы с конкретной моделью страницы. Обеспечивают проверку типов при компиляции и работу автодополнения в IDE


## Привязка значений по умолчанию
Когда HTML-хелпер генерирует поле ввода, он автоматически пытается найти для него текущее значение. Поиск значения происходит в строго определенном порядке приоритета

**Порядок приоритета значений**
  1. `ModelState` — данные, которые пользователь только что отправил (используется для сохранения введенных данных, если форма вернулась с ошибками валидации)
  2. `ViewData` или `ViewBag` — данные, переданные из контроллера через словарь
  3. Свойства объекта `Model` — данные из строго типизированной модели, привязанной к представлению


## Примеры распространенных хелперов
### Генерация ссылок
Метод `ActionLink` генерирует тег `<a>`, автоматически строя правильный URL до нужного метода контроллера на основе системы маршрутизации
```cs
@Html.ActionLink("На главную", "Index", "Home")
````

### Генерация полей ввода (input)
```CS
@model UserViewModel

@Html.TextBoxFor(m => m.Name)

@Html.PasswordFor(m => m.Password)

@Html.HiddenFor(m => m.Id)

@Html.CheckBoxFor(m => m.IsActive)
```

### Генерация выпадающего списка (select)
Для создания тега `<select>` и вложенных `<option>` используется метод `DropDownList` или `DropDownListFor`. Для их работы необходимо передать коллекцию типа `IEnumerable<SelectListItem>`

Пример подготовки данных в контроллере:
```C#
public IActionResult Create()
{
    // Создаем список элементов для выпадающего меню
    var roles = new List<SelectListItem>
    {
        new SelectListItem { Value = "1", Text = "Администратор" },
        new SelectListItem { Value = "2", Text = "Модератор" },
        new SelectListItem { Value = "3", Text = "Пользователь" }
    };

    ViewBag.RolesList = roles;
    return View(new UserViewModel());
}
```

Использование во View:
```C#
@model UserViewModel

@Html.DropDownListFor(m => m.RoleId, ViewBag.RolesList as IEnumerable<SelectListItem>, "Выберите роль", new { @class = "form-control" })
```

_Обрати внимание:_ Четвертый параметр (анонимный объект) используется для добавления произвольных HTML-атрибутов. Символ `@` перед `class` нужен, так как `class` является зарезервированным словом в C#
