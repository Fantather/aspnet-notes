Serilog принимает объект Exception и текст, который вы хотите добавить в логи


При передаче параметров *Никогда* не используйте интерполяцию строк (`$`) или `string.Format` внутри вызова логгера (`Log.XXX`)
Всегда передавайте данные как _отдельные_ параметры после строки-шаблона

**Правильно:** `Log.Information("Пользователь {UserId} вошел", user.Id);`
Мы передаём внутрь Шаблон и данные, которыми его нужно заполнить по отдельности
Благодаря этому Serilog сохраняет внутри себя структурированные данные и может с ними работать

**Неправильно:** `Log.Information($"Пользователь {user.Id} вошел")`
Serilog получил просто строку, которую он запишет, данные он так не увидит


#### Примеры
Простой пример логгера
```C#
using Serilog;

static void Main()
{
	Log.Logger = new LoggerConfiguration()
		.MinimumLevel.Debug()
		.WriteTo.File(
			"log.txt", 
			rollingInterval: rollingInterval.Day, 
			outputTemplate: "{Timestamp:yyyy-MM-dd HH:mm:ss.fff} [{level:u3}] {Message:lj}{NewLine}{Exception}"
		)
		.CreateLogger();
		
	// Теперь логгер готов к использованию
	Log.Information("Приложение запускается");
}
```

Обработчик неожиданных исключений в UI-потоке
```C#
private static void OnGuiException(object sender, ThreadExceptionEventArgs e) {
	// Используем наш логгер! Уровень "Error"
	Log.Error(e.Exception, "Поймана необработанная ошибка UI");
	
	// Показываем пользователю дружелюбное сообщение
	MessageBox.Show( "Произошла непредвиденная ошибка. Приложение попытается продолжить работу. " + "Ошибка была записана в лог.", 
		"Ошибка", MessageBoxButtons.OK, MessageBoxIcon.Error);
}
```

Обработчик не отловленных исключений из фоновых потоков
```C#
private static void OnUnhandledException(object? sender, UnhandledExceptionEventArgs e)
{
    Exception? ex = e.ExceptionObject as Exception;
    if (ex != null)
    {
        Log.Fatal(ex, "Пойман фатальный сбой (IsTerminating: {IsTerminating})", 
	        e.IsTerminating);
    }
    else
    {
        Log.Fatal("В обработчик был передан не Exception: {ExceptionObject} (IsTerminating: {IsTerminating})", 
		    e.ExceptionObject, e.IsTerminating);
    }
    
    // ВАЖНО: Принудительно сбросить логи на диск ПЕРЕД падением
    Log.CloseAndFlush();
}
```
