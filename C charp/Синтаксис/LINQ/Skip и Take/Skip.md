`Skip` - пропускает указанное количество элементов в последовательности и возвращает остальные



**Сигнатура**
`source` - исходная последовательность
`count` - число элементов, которые нужно пропустить

```C#
IEnumerable<TSource> Skip<TSource>(this IEnumerable<TSource> source, int count)
```


**Особенности**
- Если `count` превышает количество элементов, возвращается пустая последовательность
- Если `count = 0`, возвращаются все элементы
- Если `count < 0`, вызывает исключение `ArgumentOutOfRangeException`


**Пример**
```C#
var numbers = new[] { 1, 2, 3, 4, 5 };
var result = numbers.Skip(2); // Пропускаем первые 2 элемента
// Результат: { 3, 4, 5 }
```
