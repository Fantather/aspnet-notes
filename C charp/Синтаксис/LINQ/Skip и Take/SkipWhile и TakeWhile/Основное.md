- `SkipWhile` пропускает элементы, пока условие истинно, а затем возвращает все остальные
- `TakeWhile` возвращает элементы, пока условие истинно, а затем прекращает обработку

**Пример**
```C#
var numbers = new[] { 1, 2, 3, 4, 5, 6, 7 };
var skipWhile = numbers.SkipWhile(x => x < 5); // Пропускаем числа < 5
// Результат: { 5, 6, 7 }

var takeWhile = numbers.TakeWhile(x => x < 5); // Берём числа, пока они < 5
// Результат: { 1, 2, 3, 4 }
```
