Подсчитывает количество элементов  в последовательности
Может использовать предикат - что бы подсчитать количество элементов, удовлетворяющих условию

Для пустых коллекций - возвращает `0`

**Выполнение**
`IE` - Запрос выполняется Немедленно в месте своего вызова

**Перегрузки**
- `Count()` - возвращает количество элементов
- `Count(Func<TSourse, bool> predicate)` - возвращает количество элементов, удовлетворяющих условию

**Пример**
```C#
var numbers = new List<int> { 1, 2, 3, 4, 5 };
int totalCount = numbers.Count(); // 5
int evenCount = numbers.Count(n => n % 2 == 0); // 2
```

**Замечание**
Если количество элементов превысит `int.MaxValue`, выбросит `OverflowException`
Для очень больших коллекций используйте `LongCount()`
