Находит Минимальный/Максимальный элемент в последовательности

Могут работать с любыми типами, реализующими интерфейс `IComparable<T>` или кастомного `IComparer<T>`

Для пустой `не-nullable` коллекции выбрасывает `InvalidOperationException`
Для пустой `nullable` коллекции возвращает `null`

**Выполнение**
`IE` - Запрос выполняется Немедленно в месте своего вызова

**Перегрузки**
`Min()`/`Max()` - для comparable-типов
`Min/Max(Func<TSource, TResult> selector)` - применяет selector к каждому элементу и находит `min/max` среди возвращаемых значений

**Пример**
```C#
numbers.Min();
items.Max(i => i.Price);
```
