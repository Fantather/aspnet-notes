`MinBy`/`MaxBy` - возвращает элемент содержащий максимальное / минимальное значение ключа, а не просто его значение

Если последовательность пуста, возвращает `default(TSource)` (например, null для ссылочных типов)

Если несколько элементов имеют одинаковый минимальный/максимальный ключ, возвращается первый из них

Поддерживает пользовательские компараторы для сравнения ключей
