Вычисляет сумму всех числовых значений в последовательности
Может работать с любыми числовыми типами, включая `nullable` версии

Для пустой `не-nullable` коллекции возвращает `0`
Для пустой `nullable` коллекции возвращает `null`

**Выполнение**
`IE` - Запрос выполняется Немедленно в месте своего вызова

**Перегрузки**
- `Sum()` - для последовательностей числового типа
- `Sum(Func<TSourse, decimal> selector)` - вычисляет сумму значений, возвращаемых функцией selector

**Пример**
```C#
int sumNumber = numbers.Sum();
decimal totalPrice = orders.Sum(o => o.Price);
```
