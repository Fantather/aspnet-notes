`Select` - Оператор проекции, преобразует каждый элемент

**Выполнение**
`DE-S` - выдаёт элементы по мере чтения источника, без полной буферизации

**Сигнатуры**
```C#
Select(x => ...)
Select((x, i) => ...) // С индексом
```

**Примеры**
Превращает последовательность `People` в последовательность `People.Name`
```C#
people.Select(p => p.Name)
```
