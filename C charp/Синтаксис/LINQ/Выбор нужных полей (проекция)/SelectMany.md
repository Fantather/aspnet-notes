`SelectMany` - Оператор проекции, разворачивает коллекции в одну плоскую последовательность и применяет функцию к каждому элементу
Убирает только один уровень вложенности

Часто используется вместо вложенных `foreach`


**Выполнение**
`DE-S` - выдаёт элементы по мере чтения источника, без полной буферизации


**Пример**
Вернёт `IEnumerable<Item>`, которая позволит пройти по всем `Item` в каждом `Order`
```C#
class Order
{
	List<Item> Items { get; set; }
}

orders.SelectMany(o => o.Items)
```

**Примечание**
Строка - это `char-массив` так что он её раскроет и вернёт `IEnumerable<char>`
