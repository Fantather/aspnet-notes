Это оператор, который разбивает коллекцию на группы по какому-то ключу
Элементы в группах сохраняют исходный порядок последовательности

По умолчанию используется `EqualityComparer<TKey>.Default`, но можно указать другой компаратор


**Возвращаемое значение**
Возвращает набор объектов `IEnumerable<IGrouping<Tkey, TElement>>`
- `Tkey` -  тип ключа группировки
- `TElement` - тип элементов в группе

Каждая группа реализует `IEnumerable<TElement>`, поэтому можно итерироваться по её элементам

Ключ группы доступен через свойство `Key` объекта `IGrouping<TKey, TElement>`


**Выполнение**
`DE-B` - буферизующее, отложенное выполнение


**Применение**
- Анализ  данных (группировка данных категориям)
- Преобразование структур данных
- Агрегация данных (вычисление среднего, суммы и т.д. для каждой группы)
