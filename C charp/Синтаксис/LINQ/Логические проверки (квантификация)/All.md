Определяет, все ли элементы последовательности удовлетворяют условию
Для пустой последовательности возвращает `True`

**Выполнение**
**IE** (с коротким замыканием) - запрос выполняется Немедленно в месте своего вызова
Обладает `short-circuiting`(коротким замыканием) - при обнаружении элемента не удовлетворяющего условию, возвращает `false` и перебор прекращается

```C#
numbers.All(n => n % 2 == 0);
```
