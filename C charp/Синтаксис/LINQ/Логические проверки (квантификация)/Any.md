Определяет, содержит ли последовательность хотя бы Один элемент
Может принимать условие, которому должен соответствовать элемент

Значительно эффективнее `Count() > 0` для проверки на наличие элементов, особенно на потенциально больших коллекциях

**Выполнение**
**IE** (с коротким замыканием) - запрос выполняется Немедленно в месте своего вызова
Обладает `short-circuiting`(коротким замыканием) - при обнаружении элемента удовлетворяющего условию, возвращает `True` и перебор прекращается

**Перегрузки**
Возвращает `true`, если в последовательности есть хотя бы один элемент
```C#
bool isEmpty = numbers.Any();
```

Выполнение прерывается после того как предикат первый раз вернул `True`
```C#
bool isEven = numbers.Any(n => n % 2 == 0);
```
