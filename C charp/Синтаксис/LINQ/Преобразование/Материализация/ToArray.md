Немедленно материализует последовательность в массив `T[]`

Полностью перебирает исходную последовательность и создаёт массив, фиксированного размера, с элементами исходной последовательности

Обычно немного менее эффективно использует память, чем `ToList`, для коллекций, размер которых неизвестен заранее

Если исходная последовательность является массивом, создаётся его копия

**Выполнение**
`IE` - немедленное выполнение

**Пример**
Часто используется для фиксации результатов
```C#
IEnumerable<int> numbers = GetNumbers();
int[] numbersArray = numbers.ToArray();

var topProducts = products.OrderByDescending(p => p.Rating)
                         .Take(10)
                         .ToArray();
```
