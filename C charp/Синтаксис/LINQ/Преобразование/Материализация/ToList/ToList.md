Немедленно материализует последовательность в список `List<T>`

Полностью перебирает исходную последовательность и создаёт изменяемый список с элементами исходной последовательности

Если исходная коллекция уже является списком - создаётся его копия

**Использование**
Часто "замыкает" LINQ запросы
Используется для предотвращения многократного выполнения, то есть мы сохраняем результат 

**Выполнение**
- `IE` - немедленное выполнение (immediate execution)


**Пример**
```C#
IEnumerable<int> numbers = GetNumbers(); // Отложенный запрос
List<int> numbersList = numbers.ToList(); // Немедленное выполнение

// Работает с любыми IEnumerable<T>
var filtered = numbers.Where(n => n > 5).ToList();
```
