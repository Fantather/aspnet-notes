`OfType<T>()` - Возвращает только те элементы, которые можно привести к `T`

В отличии от `Cast<T>`, который пытается преобразовать все элементы в указанный тип

**Выполнение**
**DE-S** — выдаёт элементы по мере чтения источника, без полной буферизации


**Примеры**
Вернёт только объекты с типом `string`
```C#
mixed.OfType<string>()
```
