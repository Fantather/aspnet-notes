Метод `Where` - Возвращает только элементы удовлетворяющие предикату

**Выполнение**
`DE-S` - Выдаёт элементы по мере чтения источника, без полной буферизации


**Сигнатуры**
```C#
Where(x => ...)
Where((x, i) => ...) // С индексом
```


**Примеры**
Возвращает чётные числа
```C#
nums.Where(n => n % 2 == 0)
```

Возвращает числа с чётными индексами
```C#
nums.Where((num, idx) => idx % 2 == 0 
```
