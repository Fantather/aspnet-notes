```csharp
MatchCollection matches = Regex.Matches(text, pattern);
```

.NET берёт текст и **посимвольно проходит по нему**, пытаясь найти **подстроки**, которые соответствуют **шаблону (pattern)**.

---

Каждое совпадение — это объект типа `Match`, в котором хранятся:

|Свойство|Что в нём|
|---|---|
|`Value`|Что именно совпало|
|`Index`|С какой позиции в строке это началось|
|`Length`|Длина совпадения|
|`Groups`|Подгруппы (если были скобки `(...)`)|
|`Success`|Совпадение было найдено — `true/false`|

Пример:

```csharp
string pattern = @"\w+";
string text = "Слово1 и слово2.";

MatchCollection matches = Regex.Matches(text, pattern);

foreach (Match match in matches)
{
    Console.WriteLine($"Нашёл: '{match.Value}' на позиции {match.Index}");
}
```

Вывод:
```
Нашёл: 'Слово1' на позиции 0
Нашёл: 'и' на позиции 7
Нашёл: 'слово2' на позиции 9
```
