``` C#
string Str = some_var_type switch
{
	1 or 2 or 3 => "SSS",
	=> 4 and <= 7 => "AAA",
	_ => "___"
};
```

Выражение, которое сравнит `some_var_type` с имеющимися шаблонами и вернёт значение

Шаблоны

|Синтаксис|Что делает|
|---|---|
|`1`|Сравнивает с конкретным значением|
|`1 or 2 or 3`|Совпадение с одним из указанных значений|
|`>= 1 and <= 5`|Диапазон значений (с C# 9.0)|
|`_`|Подходит ко всему (аналог `default`)|
|`> 10`|Сравнение (с C# 9.0+)|
|`var x`|Сохраняет значение в переменную `x`|
|`{ Property: value }`|Сопоставление по свойствам (для объектов и рекордов)|
