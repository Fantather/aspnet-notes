`RowCount`/`ColumnCount` - количество строк и столбцов


`RowStyles`/`ColumnStyles` - тип размера
- `Absolute` - фиксированное
- `Percent` - Процент
- `AutoSize` - По содержимому


`Dock` - как `TableLayoutPanel` растягивается внутри формы. Тип `DockStyle`

| Значение       | Что делает                                                                                        |
| -------------- | ------------------------------------------------------------------------------------------------- |
| `None`         | По умолчанию. Панель не привязана ни к какой стороне и остаётся в заданных координатах и размерах |
| `Top`/`Bottom` | Панель прикрепляется к Верхней/Нижней части контейнера, растягиваясь по ширине                    |
| `Left`/`Right` | Панель прикрепляется к Левой/Правой части контейнера, растягиваясь по высоте                      |
| `Fill`         | Панель растягивается по всей ширине и высоте контейнера                                           |
| `Top, Left…`   | Можно комбинировать стороны                                                                       |


`CellBorderStyle` - будут ли видны границы ячеек внутри `TableLayoutPanel`. Тип `TableLayoutPanelCellBorderStyle`

| Значение        | Что делает                             |
| --------------- | -------------------------------------- |
| `None`          | По умолчанию. Границ нет               |
| `Single`        | Одна тонкая линия между ячейками       |
| `Inset`         | Границы с эффектом "вдавленных" клеток |
| `InsetDouble`   | Двойная вдавленная граница             |
| `Outset`        | Эффект "выпуклой границы"              |
| `OutsetDouble`  | Двойная выпуклая граница               |
| `OutsetPartial` | Частичная выпуклая граница             |
