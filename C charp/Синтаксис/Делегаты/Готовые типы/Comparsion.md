Делегат использующийся для сравнения двух значений одного типа, возвращает `int`
- `-1` если первое значение меньше второго
- `0` если значения равны
- `1` если первое значение больше второго

Ожидаемая сигнатура предиката `int(T x, T y)`

Используется в `List<T>.Sort(Comparsion<T>)`

```C#
int[] MyComparer(Comparsion<int> comp)
{
	if(comp(val1, val2))
	{
		//...
	}
}
```
