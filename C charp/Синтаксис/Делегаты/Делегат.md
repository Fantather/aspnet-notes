Делегат это ссылочный тип данных, который представляет собой ссылку(безопасный указатель) на метод(функцию) или [[Список методов|список методов]]

- Делегат инкапсулирует метод или список методов с определённой сигнатурой и позволяет вызывать их через переменную делегата

- Наследуются от `System.Delegate` и `System.MulticastDelegate`

- Содержит свойства
  - `Method` - Ссылка на вызываемый метод
  - `Target` - Ссылка на объект, если метод Экземплярный или `null` если метод Статический

Позволяет задавать логику работы метода через переданный предикат
На пример можно передать критерии сортировки

```C#
// Объявление типа делегата
public delegate int Transformer(int x);

// Использование
Transformer doubleNumber = x => x * 2; // Сохраняем в делегат Лямбду
int result = doubleNumber(5); // result == 10

// Компилятор сам создаёт делегат из метода Square, если сигнатуры совпадают
int Square(int x) => x * x;
Transformer s = Square;
int q = s(4); // q == 16
```
