**Multicast delegates**
Делегаты могут содержать _список методов_ (invocation list).
Операции `+=` и `-=` добавляют/убирают обработчики

```C#
Action a = () => Console.Writeline("one");
a += () => Console.Writeline("two");
a(); // Сначала выведет one, потом two
```

Детали:
- Для делегатов с возвращаемым значением (`Func<int, int>`) при мультивызове возвращается значение последнего метода в списке
- Если один из методов бросит исключение, то дальнейшие методы не будут вызваны, если мы не перехватим его самостоятельно
