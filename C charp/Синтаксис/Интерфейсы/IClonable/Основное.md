Интерфейс `IClonable` предназначен для создания копий объектов

Microsoft не рекомендует его использовать в новых проектах потому что:
- По умолчанию есть только `Non generic` версия
- Не известно, будет возвращена Глубокая копия или Поверхностная

Вместо этого можно
- Написать методы явного клонирования объекта
- Копирующий конструктор
- `record` типы
- Свой собственный `Generic` интерфейс


Содержит метод `Clone()` который должен возвращать новый объект, являющийся копией текущего экземпляра
