Интерфейс `IComparer` используется для сравнения объектов одного типа, если нужно несколько вариантов сравнения или если в классе не прописан `IComparable`

Содержит метод `int Compare(T x, T y)`
Пишется снаружи класса и выглядит как другой класс

```C#
public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

public class ComparerByName : IComparer<Person>
{
	public int Compare(Person x, Person y)
	{
		if (ReferenceEquals(x, y)) return 0;
		if (x is null) return -1;
		if (y is null) return 1;
		return string.Compare(x.Name, y.Name);
	}
}
```

Использование
```C#
List<Person> people = new List<Person>();
people.Sort(new ComparerByName()) // Сортировка по имени
```
