Ключевое слово `readonly` в объявлении поля `_data` означает, что это поле может быть присвоено **только один раз** — либо в момент инициализации, либо в конструкторе класса. После этого ссылка на массив `double[,]` становится неизменяемой (хотя содержимое массива всё равно можно менять).

В контексте кода:

```csharp
internal class Matrix
{
	private readonly double[,] _data;
}
```

означает, что:

- в `Matrix` нельзя будет сделать: `_data = new double[3,3];` вне конструктора;
    
- но можно менять значения внутри массива: `_data[1, 1] = 42;` — это разрешено.
    

Это полезно, чтобы предотвратить случайную замену всего массива, но при этом позволить работать с его содержимым.
