В C# метод с переменным количеством параметров создаётся с помощью ключевого слова **`params`**.

### Общая форма:
```csharp
возвращаемый_тип ИмяМетода(params тип[] имяМассива)
{
    // внутри метода все переданные значения будут доступны как массив
}
```

### Пример:
```csharp
using System;

class Program
{
    // Метод принимает любое количество целых чисел
    static int Sum(params int[] numbers)
    {
        int result = 0;
        foreach (int n in numbers)
            result += n;
        return result;
    }

    static void Main()
    {
        Console.WriteLine(Sum(1, 2, 3));        // 6
        Console.WriteLine(Sum(10, 20, 30, 40)); // 100
        Console.WriteLine(Sum());               // 0 (пустой массив)
    }
}
```

### Важные моменты:
1. **`params` может быть только один** и **только последний параметр метода**.
```csharp
void Print(string title, params string[] items) { ... }
```

2. Вызов можно делать:
- с произвольным количеством аргументов,
- или передать сразу массив:
```csharp
Print("Список", "яблоко", "груша", "банан");
Print("Список", new string[] { "яблоко", "груша" });
```

3. Можно использовать `object` как тип, если нужно принимать аргументы разных типов:
```csharp
void PrintAll(params object[] args)
{
	foreach (var arg in args)
		Console.WriteLine(arg);
}

PrintAll(1, "строка", 3.14, true);
```

---
Используется при создании метода, который будет использоваться программистами и в который они смогут передать любое количество параметров, не создавая для этого массив  
  
Например *params* есть в одной из перегрузок `Console.WriteLine(params object[] args)` (У object есть `ToString()`, если что)  
  
Можно использовать в методах для математических вычислений
