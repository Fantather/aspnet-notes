Означает доступ по индексу
В C# у него как и у Свойств есть Аксессоры - `get` и `set`

Индексаторы (`this[...]`) дают возможность обращаться к экземпляру класса как к массиву: `obj[index]`


Цели:
- Предоставить удобный доступ к внутренним данным (коллекции, матрицы, таблицы)
- Скрыть внутреннюю структуру (например, словарь) и дать чистый API
- Поддержать несколько параметров в индексаторе (например, двумерный индекс `matrix[i, j]`)

Пример индексатора - матрица:
```csharp
public class Matrix
{
    private readonly double[,] _data;
    public int Rows { get; }
    public int Cols { get; }

    public Matrix(int rows, int cols)
    {
        Rows = rows; Cols = cols;
        _data = new double[rows, cols];
    }

    // Индексатор с двумя параметрами
    public double this[int row, int col]
    {
        get => _data[row, col];
        set => _data[row, col] = value;
    }
}
```

Преимущества:
- Код клиента: `matrix[2,3] = 5.0;` - очень читабельно.
- Можно выполнять валидацию индексов внутри индексатора
