Используется для проверки соответствия типа объекта
Проверяет, можно ли привести тип к определённому типу или реализует ли он интерфейс

```C#
if (obj is SomeType)
{
    // obj можно безопасно привести к SomeType
}
```

`C# 7.0` и выше позволяет сразу извлекать значение нужного типа
```C#
if(obj is string s)
{
	Console.Writeline(s);  // s - значение из obj приведённое к string
}
```

Проверка на `null`
```C#
object obj = null;
Console.WriteLine(obj is null);  // true
```
