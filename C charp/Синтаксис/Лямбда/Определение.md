Безымянная функция которую можно присвоить Делегату или передать как параметр методу
Нужны, если подобная функция мне больше нигде в коде не пригодится

Лямбда автоматически преобразуется в совместимый делегат или `Expression<Func<...>>`, если метод этого ожидает


**Синтаксис**
Однострочное выражение
```C#
x => x * x;  // Берёт x и возвращает x²
```

Несколько входных параметров
```C#
(a, b) => a + b;  // Суммирует a+b
```

Многострочное выражение
```C#
number => {
	if(number % 2 == 0)
		return "Even";
		
	else
		return "Odd";
}
```
