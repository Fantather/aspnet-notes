`System.Object` - является родительским для каждого типа в [[CLR]] языках программирования, согласно [[CTS]]

То есть каждый тип данных в CLR умеет
- Сравнить два экземпляра на равенство
  - C#, `Equals(obj)`

- Получить хеш-код экземпляра
  - C#, `GetHash()`

- Запросить фактический тип экземпляра
  - C#, `typeof()`

- Выполнить поверхностное копирование экземпляра
  - C#, `Copy()`

- Получить строковое представление текущего состояния экземпляра
  - C#, `ToString()`
