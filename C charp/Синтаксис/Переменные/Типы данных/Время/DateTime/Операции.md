**Создание:**
```C#
var now       = DateTime.Now;          // текущее локальное время
var utcNow    = DateTime.UtcNow;       // текущее UTC
var specific  = new DateTime(2025,7,13,14,30,0);  // 13 июля 2025, 14:30:00, Kind=Unspecified
var fromTicks = new DateTime(637676928000000000L, DateTimeKind.Utc);
```


**Изменение даты:**
```C#
DateTime tomorrow = now.AddDays(1);
```
Добавляем день

``` C#
DateTime minusTwoHours = now.AddHours(-2);
```
Отнимаем два часа

``` C#
TimeSpan duration = tomorrow - now;     // TimeSpan = 1 day
```
