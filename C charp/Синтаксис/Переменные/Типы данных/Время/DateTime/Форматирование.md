1. Пользовательский формат: `"2025-07-13 14:30:00"`
``` C#
string s1 = now.ToString("yyyy-MM-dd HH:mm:ss");
```
- `yyyy` — 4‑значный год (2025)
- `MM` — 2‑значный месяц (07)
- `dd` — 2‑значный день (13)

- `HH` — часы в 24‑часовом формате (14)
- `mm` — минуты (30)
- `ss` — секунды (00)


2. Стандартный формат с учётом культуры: `"13 июля 2025 г"`
```C#
string s2 = now.ToString("D", CultureInfo.CreateSpecificCulture("ru-RU"));
```

- `"D"` — это стандартный формат даты `Long date pattern` (длинный формат)
- `CultureInfo.CreateSpecificCulture("ru-RU")` задаёт "русскую" культуру

Другие примеры стандартных паттернов:
- `"d"` — короткий формат (для `ru-RU`: `"13.07.2025"`)
- `"T"` — длинное время (`"14:30:00"`)
- `"g"` — общее короткое (`"13.07.2025 14:30"`)
