Тип представляющий интервал времени, а не момент времени

**Инициализация**
```C#
TimeSpan t1 = TimeSpan.FromSeconds(5);        // 5 секунд
TimeSpan t2 = new TimeSpan(0, 1, 30);         // 0ч, 1м, 30с
TimeSpan t3 = TimeSpan.FromMilliseconds(250)  // 250мс
TimeSpan zero = TimeSpan.Zero;                // 0ч, 0м, 0с
TimeSpan inifinite = Timeout.InfiniteTimeSpan;// Бесконечный период
```

- `TimeSpan.Zero` - Константа обозначающая *нулевую длительность*, начало отсчёта с нуля
- `Timeout.InfiniteTimeSpan` - Это константа обозначающая бесконечный интервал ожидания
  Хранит `-1` миллисекунду
  Во многих API отрицательное время означает Ждать бесконечно


**Компоненты времени**
```C#
var t = new TimeSpan(1, 2, 30);    // 1 час, 2 минуты, 30 секунд
Console.WriteLine(t.TotalMinutes); // 62.5 (всего минут)
Console.WriteLine(t.TotalSeconds); // 3750 (всего секунд)
Console.WriteLine(t.TotalHours);   // 1.041666...
```
- `TotalX` - свойство, общее время выраженное в выбранных единицах времени
  - `TotalMilliseconds`
  - `TotalSeconds`
  - `TotalMinutes`
  - `TotalHours`
  - `TotalDays`

```C#
var t = new TimeSpan(1, 2, 30);// 1 час, 2 мин, 30 сек
Console.WriteLine(t.Hours);    // 1, но если бы было 25 часов, выдало бы 1, а не 25
Console.WriteLine(t.Minutes);  // 2
Console.WriteLine(t.Seconds);  // 30
```
- Если интервал больше суток, `Hours` будет только остатком в пределах 0–23
- Если больше минуты, `Seconds` будет остатком в пределах 0–59

**TimeSpan.ToString()**
В виде строки `TimeSpan` выглядит как
```C#
[знак -][дни.]часы:минуты:секунды[.доли_секунд]
```
- `дни.` - выводятся только если интервал ≥ 1 день
- `часы` - всегда две цифры (00–23, остаток в пределах суток)
- `минуты` - 00–59
- `секунды` - 00–59
- `.доли_секунд` - выводятся, если они не равны нулю
  Точность до 7 знаков после точки (точность `Ticks`)
- Если время отрицательное - в начале будет `-`

```C#
Console.WriteLine(TimeSpan.FromSeconds(45));       // 00:00:45
Console.WriteLine(TimeSpan.FromMinutes(5));        // 00:05:00
Console.WriteLine(TimeSpan.FromHours(1.5));        // 01:30:00
Console.WriteLine(TimeSpan.FromDays(2));           // 2.00:00:00
Console.WriteLine(TimeSpan.FromMilliseconds(123)); // 00:00:00.1230000
Console.WriteLine(TimeSpan.FromSeconds(-5));       // -00:00:05
Console.WriteLine(new TimeSpan(1, 2, 3, 4, 567));  // 1.02:03:04.5670000
```
