**Свойства**
- `Interval` (double, миллисекунды)
  Интервал между срабатываниями

- `AutoReset` (Bool)  - по умолчанию `true`
  - `true` - После срабатывания таймер будет автоматически перезапущен
  - `false` - Таймер сработает один раз и остановится, для повторного запуска можно вызвать `Start()`

- `Enabled` (Bool)
  - `true` - Запускает таймер, эквивалент `Start()`
  - `false` - Останавливает таймер, эквивалент `False()`
  
- `Elapsed`
  Событие, вызываемое при срабатывании
  На него подписываются обработчики
  
- `SynchronizingObject`
  Объект, на который будут маршалиться события (Обычно используется в `WinForms`)

**Методы**
- `Start()`, `Stop()` - управление работой таймера
- `Dispose()` - освобождение ресурсов


В `System.Timers.Timer` нет метода `Reset()`
Что бы остановить таймер и вернуть его к началу можно сделать так
```C#
timer.Stop();              // или timer.Enabled = false;
// Можно изменить Interval
timer.Start();             // или timer.Enabled = true;
```
