```C#
Dictionary<string, int> dict = [];

dict["lemon"] = 10;
```

`dict[key]` - Индексатор перезаписывает значение, если ключ уже есть в Словаре или создаёт новую пару `Ключ-Значение`

**Через метод**
Если передаваемый ключ уже есть в словаре, то вызовется ошибка `ArgumentException`
```C#
dict.Add("orange", 3);
```

**Безопасное добавление**
`TryAdd` вернёт `False` если ключ уже есть
```C#
dict.TryAdd("bird", 23);
```
