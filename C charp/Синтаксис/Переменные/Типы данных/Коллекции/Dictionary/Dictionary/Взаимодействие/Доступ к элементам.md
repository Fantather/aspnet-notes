**TryGetValue**
Обычно он используется для доступа к Словарю
```C#
if (dict.TryGetValue("grapes", out int value))
	Console.WriteLine(value);

else
	Console.WriteLine("Такого ключа нет");
```


**Индексатор**
При использовании Индексатора для доступа, если указанного ключа не будет в словаре, то будет вызвана ошибка `KeyNotFoundException`
```C#
Dictionary<string, int> dict = [];

// Если ключа не будет, вызовется ошибка KeyNotFoundException
int value = dict[key]; 
```


**GetValueOrDefault**
`GetValueOrDefault(key, DefaultValue)` - Означает что если ключ будет найден, то вернётся значение, если нет - то указанное значение по умолчанию

```C#
dict.GetValueOrDefault("apple", 0);
```
