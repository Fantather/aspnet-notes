`Count` - Свойство, показывает количество элементов

`Dequeue()` - Удаляет и возвращает элемент из начала очереди
`Peek()` - Возвращает элемент из начала очереди, не удаляя его
`Enqueue(T item)` - Добавляет элемент в конец очереди


```C#
Queue<string> queue = new Queue<string>();

queue.Enqueue("Первый");
queue.Enqueue("Второй");

Console.WriteLine(queue.Peek());  // "Первый", но не удаляет
Console.WriteLine(queue.Dequeue()); // "Первый", удаляется
Console.WriteLine(queue.Dequeue()); // "Второй", удаляется
```
