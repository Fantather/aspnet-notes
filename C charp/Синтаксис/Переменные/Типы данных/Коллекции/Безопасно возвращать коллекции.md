1. Возвращать копию массива
   Тогда изменения не затронут оригинальный список

2. Возвращать `IEnumerable<T>`
   Абстрактный вариант, доступен только перебор
   Возвращать интерфейс с конкретным типом (`IEnumerable<int>`)

3. Вместо свойства с массивом, сделать методы для доступа, на пример через индексатор 

4. Возвращать интерфейс `IReadOnlySomething<T>`
   Для каждого типа он работает по своему, на пример `IReadOnlyList<string>`
   Снаружи можно будет читать, но не изменить коллекцию (`Add`, `Remove` недоступны), но изменить Содержимое коллекции **можно**, поэтому это безопасно только с неизменяемым типом `string`
   
   Если будет содержать неизменимые типы, такие как Структуры или `record`, то изменить их тоже будет нельзя
   
   А в `IReadOnlyDictionary<>` нельзя изменять значения по ключу 


**Примеры**
1. Возвращение копии массива
```C#
public class Example
{
	private readonly List<string> _collection = new();
	
	// Короткая запись
	public List<string> Collection => new (_collection);
	
	// Длинная запись
	public List<string> Collection
	{
		get
		{
			return new List<string>(_collection);
		}
	}
}
```

2. Возвращать `IEnumerable<T>`
```C#
public class Example
{
	private readonly List<string> _collection;
	
	// Короткая запись
	public IEnumerable<string> Collection => _collection;
	
	// Длинная запись
	public IEnumerable<string> Collection
	{
		get
		{
			// У `List<T>` нет метода `GetEnumerable()`
			// он сам реализует `IEnumerable<T>`
			return _collection;
		}
	}
}
```

3. Методы доступа
```C#
public class Example
{
	private readonly List<string> _collection = new();
	public void Add(string item) => _collection.Add(item);
	public void Count() => _collection.Count();
	public sting this[int index]
	{
		get
		{
			return _collection[index];
		}
		set
		{
			_collection[index] = value;
		}
	}
}
```

4. Возвращать `IReadOnlyList<string>`
```C#
public class Example
{
	private readonly List<string> _collection;
	
	// Короткая запись
	public IReadOnlyList<string> Collection => _collection;
	
	// Длинная запись
	public IReadOnlyList<string> Collection
	{
		get
		{
			return _collection;
		}
	}
}
```
