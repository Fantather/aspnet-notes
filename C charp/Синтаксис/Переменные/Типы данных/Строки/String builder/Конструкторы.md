
| Сигнатура                                   | Что делает                                                                  |
| ------------------------------------------- | --------------------------------------------------------------------------- |
| `StringBuilder()`                           | Пустой буфер, `Capacity` по умолчанию (~16 символов)                        |
| `StringBuilder(int capacity)`               | Инициализирует буфер указанного размера                                     |
| `StringBuilder(string value)`               | Копирует символы `value`, устанавливает `Capacity = value.Length + default` |
| `StringBuilder(string value, int capacity)` | То же, но с явным указанием `Capacity`                                      |


**Пример использования**

```csharp
var sb = new StringBuilder(50);   // заранее резервируем место
sb.Append("ID: ");
sb.Append(123);
sb.Append(", Name: ");
sb.AppendLine("Alice");

// Вставим дату в начало
sb.Insert(0, DateTime.Now.ToString("yyyy-MM-dd") + " — ");

// Заменим запятую на точку с запятой
sb.Replace(',', ';');

// Получим готовую строку
string result = sb.ToString();
Console.WriteLine(result);
```


**Когда использовать**
- Большое количество последовательных конкатенаций (например, в цикле).
- Динамическое построение больших текстов (HTML, SQL, логи).
- Когда важна производительность и вы хотите избежать лишних аллокаций строк.

Если вы просто конкатенируете пару строк, обычный оператор `+` или интерполяция ( `$"{a}{b}"` ) удобнее и чище.
