## Поля
| Член                                                | Описание                                                                       |
| --------------------------------------------------- | ------------------------------------------------------------------------------ |
| `int Length { get; set; }`                          | Текущая длина (количество символов)                                            |
| `int Capacity { get; set; }`                        | Размер внутреннего буфера (≥ `Length`)                                         |

## Методы

| Член                                                | Описание                                                                       |
| --------------------------------------------------- | ------------------------------------------------------------------------------ |
| `StringBuilder Append(...)`                         | Добавляет данные в конец: строки, символы, числа, объекты (через `ToString()`) |
| `StringBuilder AppendLine(...)`                     | То же + добавляет символы новой строки                                         |
| `StringBuilder Insert(int index, ...)`              | Вставляет в указанную позицию                                                  |
| `StringBuilder Remove(int startIndex, int length)`  | Удаляет диапазон символов                                                      |
| `StringBuilder Replace(char oldChar, char newChar)` | Заменяет все вхождения символа                                                 |
| `override string ToString()`                        | Собирает содержимое буфера в новый объект `String` и возвращает его            |
| `void Clear()`                                      | Очищает содержимое, оставляя буфер неизменным                                  |

> **Важно:** все методы `Append`/`Insert`/**`Replace`** возвращают тот же самый экземпляр `StringBuilder` — это позволяет строить цепочки (fluent interface):

``` c#
var sb = new StringBuilder()
     .Append("Hello, ")
     .Append(name)
     .AppendLine("!");
```
