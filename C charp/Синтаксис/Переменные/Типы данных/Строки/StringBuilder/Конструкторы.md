- `new StringBuilder()` - стартовая ёмкость по умолчанию (≈16 символов).
- `new StringBuilder(int capacity)` - сразу резервирует указанный размер.
- `new StringBuilder(string value)` — инициализирует содержимым `value`.
- `new StringBuilder(string value, int capacity)` — задаёт начальное содержимое и ёмкость.
