
| Метод                        | Описание                                       |
| ---------------------------- | ---------------------------------------------- |
| `Append(...)`                | Добавляет строку/символ/другой `StringBuilder` |
| `AppendLine(...)`            | Добавляет в конец символ новой строки          |
| `AppendFormat(...)`          | Для вставки форматированных данных             |
| `Insert(int index, ...)`     | Вставляет в указанную позицию                  |
| `Replace(old, @new)`         | Заменяет все вхождения `old` на `new`          |
| `Remove(int start, int len)` | Удаляет указанный фрагмент                     |
| `Clean()`                    | Очищает буфер (аналогично `Length = 0`)        |
| `ToString()`                 | Возвращает объект типа `string`                |
| `EnsureCapacity(int cap)`    | Устанавливает минимальный размер буфера        |
