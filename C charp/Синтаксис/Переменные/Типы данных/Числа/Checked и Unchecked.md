### **checked**
- **Цель:** Включает проверку переполнения при арифметических операциях с числами.

- Если число выйдет за пределы типа — будет выброшено `OverflowException`.

```C#
checked
{
	int x = int.MaxValue + 1; // будет исключение
}
```

---

### 9. **unchecked**

- Отключает проверку переполнения.
- Переполнение происходит silently (тихо), число "оборачивается"
  В случае примера, `x = int.MinValue`


```C#
unchecked
{
	int x = int.MaxValue + 1; // переполнение без исключения
}
```
