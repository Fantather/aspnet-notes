Это структура предназначенная для хранения десятичный дробей без накопления двоичных ошибок, подходит для финансов, весов, процентов
Внутри структура хранит 96‑битное целое значение и 8‑битный «экспонент» (масштаб), плюс бит знака

Литерал:
```C#
decimal price = 123.45m;  // суффикс m или M обязателен
```
