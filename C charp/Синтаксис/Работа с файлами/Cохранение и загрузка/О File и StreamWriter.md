
`File` - Статический класс, высокоуровневый инструмент, предназначенный для быстрых и простых операций (Прочитать/Записать весь файл сразу)

- Удобен и прост
  Одна строка кода для чтения или записи всего содержимого
  
- Работает со всем файлом целиком
  Методы `ReadAllText`, `ReadAllLines`, `WriteAllText`, `WriteAllLines` загружают весь файл в память или записывают все данные из памяти на диск - одной операцией
  
- Нет контроля над процессом
  Вы не можете контролировать, как происходит чтение или запись (На пример, обрабатывать данные по частям по мере чтения)
  Файл либо полностью прочитан, либо полностью записан


`StreamWriter`/`StreamReader` - это низкоуровневые инструменты для тонкой работы, они дают полный контроль над процессом
Эти классы работают с файлами как с потоком данных

`StreamWriter` - предназначен для Записи текстовых данных в поток
Наследуется от `TextWriter`

`StreamReader` - предназначен для Чтения текстовых данных из потока
Наследуется от `TextReader`


- Поточная обработка
  Данные читаются или пишутся последовательно, по частям
  Это не требует загрузки всего файла в память
- Контроль и гибкость
  Вы можете читать файл построчно (`ReadLine`), посимвольно, до конца('ReadToEnd') или записывать данные постепенно добавляя их (`Write`, `WriteLine`)
- Буферизация
  По умолчанию эти классы буферизованы, то есть, данные сначала накапливаются в буфере памяти, а затем записываются на диск большими блоками
  Это повышает производительность при множественных, мелких операциях записи
- Эффективность работы с большими файлами
  Главное преимущество
  Есть возможность обрабатывать файлы Гигабайтных размеров, не потребляя много оперативной памяти


Отлично подходят для 
- Обработки больших файлов, которые не помещаются в память
- Сценариев, где нужно постепенно обрабатывать данные
  На пример, читать лог-файл, анализировать каждую строку и сразу же выводить результат
- Постепенного добавления данных в файл
  На пример, логгирование работы приложения с помощью `StreamWriter`, который удобно создается через метод `File.AppendText()`
- Работы с сетевыми потоками или другими источниками данных, отличными от файлов
