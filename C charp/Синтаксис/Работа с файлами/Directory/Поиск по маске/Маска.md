Поиск файлов по маске - поиск файлов не по точному имени. а по шаблону
- `*.txt` - Все текстовые файлы
- `data*.json` - Все файлы, которые начинаются на `data` и заканчиваются на `.json`
- `report_??.csv` - Все файлы формата `report_01.csv`, `report_99.csv` (где `?` заменяет ровно один символ)

Для этого используется метод `GetFiles`
