**Удаление файла**
```C#
string path = @"C:\Temp\test.txt";

if(File.Exists(path))   // Проверяем, существует ли файл
{
	File.Delete(path);  // Удаляем
}

else
	Console.WriteLine("Файл не найден");
```


**Удаление нескольких файлов по маске**
Для этого сначала используем `GetFile` для нахождения пути к файлу по маске, а потом удаляем их
```C#
string path = @"C:\Temp";
string mask = "*.log"     // Удалим все log файлы

string[] files = Directory.GetFiles(path, mask);

foreach(string file in files)
	File.Delete(file);
```
