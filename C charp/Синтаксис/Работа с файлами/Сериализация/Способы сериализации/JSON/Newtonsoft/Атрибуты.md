**Имя свойства**
В коде будет `FullName`, в JSON будет `Customer Name`
```C#
 [JsonProperty("Customer Name")]
 public string FullName { get; set; }
```

**Порядок свойств**
Задает порядок свойств, в котором свойства будут выводиться в JSON
```C#
[JsonProperty(Order = 1)]
```

**NullValueHandling**
Если значение свойства `null`, оно будет полностью пропущено в результирующем JSON
```C#
[JsonProperty(NullValueHandling = NullValueHandling.Ignore)]
```


**JsonIgnore**
Помечает свойство, которое должно быть полностью проигнорировано и при сериализации (не попадёт в JSON) и при десериализаци (даже если оно есть в JSON, будет проигнорировано)
```C#
[JsonIgnore]
```
