`NewtonSoft.JSON`/`Json.NET` - самая распространённая библиотека в мире .NET, хотя Microsoft сейчас активно продвигает свой аналог


|Характеристика|System.Text.Json|Newtonsoft.Json|
|---|---|---|
|**Производительность**|**Выше** (оптимизирована для .NET, использует `Span<T>`, меньше аллокаций памяти)|Ниже|
|**Безопасность**|Более строгие настройки по умолчанию (например, не разрешает десериализацию небезопасных типов)|Более либеральная по умолчанию|
|**Встроенность**|Поставляется с .NET (начиная с .NET Core 3.0), не требует сторонних зависимостей|Сторонняя библиотека (NuGet)|
|**Функциональность**|Изначально была менее функциональна, но быстро догоняет (особенно в .NET 7 и 8)|Очень богатая и зрелая функциональность|
