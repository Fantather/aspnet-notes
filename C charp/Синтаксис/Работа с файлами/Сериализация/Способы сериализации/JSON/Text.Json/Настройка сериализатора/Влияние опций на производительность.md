Различные опции `JsonSerializerOptions` по-разному влияют на производительность:

- **Высокое влияние**: `ReferenceHandler.Preserve`, `IncludeFields`, кастомные конвертеры

- **Среднее влияние**: `PropertyNameCaseInsensitive`, `PropertyNamingPolicy`

- **Низкое влияние**: `WriteIndented`, `IgnoreNullValues`
