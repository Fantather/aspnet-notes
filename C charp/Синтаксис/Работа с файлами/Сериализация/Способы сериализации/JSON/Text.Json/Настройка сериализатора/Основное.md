**JsonSerializerOptions** - это класс в пространстве имён `System.Text.Json`, который предоставляет параметры для настройки поведения сериализатора и десериализатора JSON
Он позволяет контролировать различные аспекты процесса преобразования данных, такие как политики именования, обработка null-значений, форматирование вывода и многое другое

Использование этого класса особенно при необходимости тонкой настройки процесса сериализации/десериализации под конкретные требования приложения

Если в сохранении были опции, которые влияют на сохранённые данные, на пример `IncludeFields`, то при загрузке этого файла, такие же опции тоже должны быть переданы

Для **оптимизации производительности** рекомендуется сохранять в переменные и повторно использовать экземпляры JsonSerializerOptions, особенно при частых вызовах сериализации/десериализации, так как они кэшируют метаданные типов, это тяжёлый процесс
