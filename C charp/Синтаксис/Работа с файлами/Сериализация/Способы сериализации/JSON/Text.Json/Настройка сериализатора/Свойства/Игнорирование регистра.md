`PropertyNameCaseInsensitive` - определяет, следует ли игнорировать регистр символов при сопоставлении имён свойств во время десериализации
По умолчанию - `false`

```C#
internal class Person
{
	public string Name { get; set; }
	public int Age { get; set; }
}

var options = new JsonSerializerOptions
{
	PropertyNameCaseInsensitive = true
};


// Десериализация будет успешной, несмотря на разный регистр
string json = "{\"nAmE\": \"John\", \"aGe\": 30}";
Person person = JsonSerializer.Deserialize<Person>(json, options);
```
