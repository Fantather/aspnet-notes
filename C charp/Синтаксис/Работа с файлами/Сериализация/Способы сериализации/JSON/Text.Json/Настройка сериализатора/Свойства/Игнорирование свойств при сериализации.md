`DefaultIgnoreCondition` - Определяет условия, при которых свойства игнорируются при сериализации

Позволяет гибко управлять тем, какие свойства включать в сериализацию
Заменяет устаревшее `IgnoreNullValues`

- `Never` - свойства никогда не игнорируются
- `Always` - свойства всегда игнорируются
- `WhenWritingNull` - свойство не сериализуется, если его значение `null`
- `WhenWritingDefault` - свойство не сериализуется, если у него значение по умолчанию (например, `0` для чисел, `false` для `bool`, `null` для ссылочных типов)

```C#
var options = new JsonSerializerOptions
{
    DefaultIgnoreCondition = JsonIgnoreCondition.WhenWritingNull
};

var person = new Person { Name = "John", Age = null };
// Сериализует { "Name": "John" } без свойства Age
string json = JsonSerializer.Serialize(person, options);
```


`IgnoreNullValues` - устаревшая опция, определяет, следует ли игнорировать свойства с значениями `null` при сериализации
Устарело в пользу `DefaultIgnoreConditions`

```C#
var options = new JsonSerializerOptions
{
    IgnoreNullValues = true // Устаревшее, но еще поддерживаемое
};

var person = new Person { Name = "John", Age = null };
// Сериализует { "Name": "John" } без свойства Age
string json = JsonSerializer.Serialize(person, options);
```
