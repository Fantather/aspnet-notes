`MaxDepth` - определяет максимальную глубину вложенности объектов/массивов при сериализации и десериализации

По умолчанию значение `0`, оно разрешает 64 уровня вложенности
Если объект окажется глубже - сериализация/десериализация выбросит `JsonException` 

Используется для
- Предотвращения атак глубокой рекурсией ([[JSON - бомба]])
  
- Контроля сложности данных
  В реальных системах часто известно, что глубина не должна быть больше какой-то разумной
  Тогда можно явно задать `MaxDepth = 10` и гарантировать, что при ошибках или неожиданной структуре будет выброшено исключение
  
- Предотвращение переполнения стека
  При рекурсивной сериализации слишком глубокая структура может привести к `StackOverflowException`
  Ограничение `MaxDepth` останавливает этот процесс

```C#
var options = new JsonSerializerOptions
{
    MaxDepth = 32 // Ограничить глубину вложенности, до 32
};

// Выбросит исключение, если глубина превышает 32
try
{
    string json = JsonSerializer.Serialize(deepObject, options);
}
catch (JsonException ex)
{
    // Обработка ошибки
}
```
