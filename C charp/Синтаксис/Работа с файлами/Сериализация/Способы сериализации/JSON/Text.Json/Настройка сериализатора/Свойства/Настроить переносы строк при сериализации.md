`WriteIndented` - определяет, следует ли форматировать выходной JSON с отступами и переносами строк для улучшенной читаемости
Позволяет настраивать отступы при сериализации (с .NET 8)
По умолчанию - `false`, что бы не увеличивать размер файла

Используется при генерации человекочитаемого JSON для логирования, конфигурационных файлов или ответов API в разработке

Обычное использование
```C#
var options = new JsonSerializerOptions
{
    WriteIndented = true
};

// Сериализует с отступами:
// {
//   "Name": "John",
//   "Age": 30
// }
string json = JsonSerializer.Serialize(person, options);
```


Настройка отступов (в .NET 8 и новее)
```C#
var options = new JsonSerializerOptions
{
    WriteIndented = true,
    IndentCharacter = '\t', // Использовать табуляцию вместо пробелов
    IndentSize = 2          // Размер отступа
};
```
