`IncludeFields` - Определяет, следует ли сериализовать и десериализовать поля (в дополнение к свойствам)
По умолчанию - `false`

Используется при работе с типами, где данные хранятся в полях, а не в свойствах

```C#
var options = new JsonSerializerOptions
{
    IncludeFields = true
};

public class Person
{
    public string Name;
    public int Age;
}

// Сериализует и поля тоже
string json = JsonSerializer.Serialize(person, options);
```
