`NumberHandling` - определяет, как обрабатываются числовые типы при сериализации и десериализации

Используется для работы с API, которые представляют числа в виде строк (На пример, для точности больших чисел)

Доступные флаги:
- `Strict` - стандартное поведение
- `AllowReadingFromString` - разрешить  чтение чисел из строк
- `WriteAsString` - записывать числа как строки

```C#
var options = new JsonSerializerOptions
{
    NumberHandling = JsonNumberHandling.AllowReadingFromString | 
                    JsonNumberHandling.WriteAsString
};

// Десериализует JSON с числами в строках: {"age": "30"}
var person = JsonSerializer.Deserialize<Person>(json, options);

// Сериализует числа как строки: {"age": "30"}
string json = JsonSerializer.Serialize(person, options);
```
