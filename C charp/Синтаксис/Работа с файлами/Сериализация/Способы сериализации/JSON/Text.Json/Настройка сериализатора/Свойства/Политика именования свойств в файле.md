`PropertyNamingPolicy` - Определяет политику именования свойств
Можно использовать встроенные политики или создавать собственные

Использовать camelCase
```C#
var options = new JsonSerializerOptions
{
	PropertyNamingPolicy = JsonNamingPolicy.CamelCase
};

// Сериализует { "name": "John", "age": 30 } вместо { "Name": "John", "Age": 30 
string json = JsonSerializer.Serialize(person, options);
}
```

Пользовательское именование
```C#
public class UpperCaseNamingPolicy : JsonNamingPolicy
{
    public override string ConvertName(string name) => name.ToUpper();
}

var options = new JsonSerializerOptions
{
    PropertyNamingPolicy = new UpperCaseNamingPolicy()
};
// Сериализует { "NAME": "John", "AGE": 30 }
```
