`Encoder` - Позволяет задать кодировщик для управления экранированием символов
Особенно полезно для обработки не-ASCII символов

Используется для обработки специальных символов, Unicode-символов или предотвращения экранирования определённых символов



Разрешаем в JSON латиницу (Basic Latin: `U+0000–U+007F`, т.е. английский алфавит, цифры, знаки пунктуации)
Разрешаем кириллицу (Cyrillic: `U+0400–U+04FF` и т.д.)

Все остальные символы (например, китайские иероглифы, арабские буквы, эмодзи) будут **закодированы в `\uXXXX`**, чтобы JSON был «безопасен» и кросс-платформенно читаем

```C#
var options = new JsonSerializerOptions
{
    Encoder = JavaScriptEncoder.Create(UnicodeRanges.BasicLatin, UnicodeRanges.Cyrillic)
};

var person = new Person { Name = "Иван", Age = 30 };
// Сериализует { "Name": "Иван", "Age": 30 } без экранирования кириллицы
string json = JsonSerializer.Serialize(person, options);
```


`UnicodeRanges` - это набор Предопределённых диапазонов символов Unicode, которые предоставляет .NET для управления тем, какие символы будут экранироваться при сериализации/кодировании строк

По сути, это способ сказать энкодеру, какие символы разрешено выводить как есть, а какие надо закодировать в `\uXXXX`
