Простой класс для сериализации
```C#
public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
    public DateTime Birthday { get; set; }
}

```


```C#
static void Main()
{
	Person person = new () { Name = "Alice", Age = 30, Birthday = new DateTime(1993, 5, 10)};
	
	string jsonString = JsonSerializer.Serialize(person);
	
	Person deserializedPerson = JsonSerializer.Deserialize<Person>(jsonString);
}
```
Создаём объект
Сериализуем объект в строку JSON
Десериализуем строку обратно в объект

`Deserialize` может вернуть `null`, если переданная строка будет
- `null`
- Пустой
- Содержать только пробелы

В остальных случаях `Deserialize` либо
- Успешно создаст объект, даже если не все свойства совпадают
  Не совпавшие становятся значениями по умолчанию
- Либо выбрасывает `JsonException`, если JSON синтаксически не корректный


**Сохранение и загрузка**
