Это специальные метаданные, которые вы добавляете к классам, свойствам и полям, чтобы явно указать сериализатору как именно нужно преобразовывать объект

Без атрибутов сериализатор использует поведение по умолчанию
Называет поля так же, как свойство, включает все публичные свойства в файл и т.д.
Атрибуты дают вам точный контроль над этим процессом

```C#
using System.Text.Json.Serialization;

public class Product
{
    // Свойство будет называться "product_name" в JSON
    [JsonPropertyName("product_name")]
    public string Name { get; set; }

    // Свойство будет полностью игнорироваться при сериализации
    [JsonIgnore]
    public decimal Price { get; set; }

    // Можно управлять форматом для DateTime
    [JsonConverter(typeof(JsonStringEnumConverter))] // Сериализует enum как строку, а не число
    public ProductCategory Category { get; set; }
}

public enum ProductCategory { Food, Electronics, Books }
```
