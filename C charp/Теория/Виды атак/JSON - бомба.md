Злоумышленник может прислать JSON с очень глубокой вложенностью:

```C#
{ "a": { "b": { "c": { "d": { ... } } } } }
```

На первый взгляд это небольшой JSON, но при глубине в десятки тысяч уровней сериализатор начнёт тратить много памяти и процессорного времени → приложение "ляжет".  
Ограничение по `MaxDepth` спасает от этого
