*Virtual Execution System* - часть [[CLI]],
это описание виртуальной исполняющей системы, которая обеспечивает среду выполнения для программ, скомпилированных в [[IL-код|CIL]] (Common Intermediate Language).

Она отвечает за
- загрузку и выполнение кода (В [[CLR]] это делает [[JIT]])
- управление памятью во время выполнения (через GC, хотя сам GC - отдельный компонент)
- проверку типов
- обработку исключений
- безопасность
- взаимодействие между языками
