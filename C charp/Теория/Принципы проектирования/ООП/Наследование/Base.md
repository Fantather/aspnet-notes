Ключевое слово `base` предоставляет доступ к членам базового класса из производного класса

1. *Вызов конструктора базового класса*: Как показано выше, `: base(...)`
2. *Вызов методов базового класса*: Когда производный класс переопределил метод (`override`), но внутри своей реализации хочет вызвать оригинальную версию метода из базового класса
3. *Доступ к скрытым членам*: Если производный класс *скрыл* член базового класса (с помощью `new`), `base` позволяет обойти [[Сокрытие имён при наследовании|сокрытие]] и получить доступ к оригинальному члену
