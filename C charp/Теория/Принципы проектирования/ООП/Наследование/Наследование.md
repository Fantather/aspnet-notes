*Наследование* - это механизм объектно-ориентированного программирования (ООП), который позволяет одному классу (дочернему или производному) наследовать поля, свойства, методы и другую функциональность другого класса (родительского или базового)

- **Цель:** Повторное использование кода (избежание дублирования), создание иерархии классов и полиморфизм

- **Синтаксис в C#:** Дочерний класс указывается через двоеточие `:` после имени, за которым следует имя базового класса

- **Отношение:** Наследование выражает отношение **«is-a»** (является)
  Например, класс `Student` наследует от `Person`, потому что студент _является_ человеком

```C#
// Базовый класс
public class Person
{
    public string Name { get; set; }
    public void Walk()
    {
        Console.WriteLine($"{Name} is walking.");
    }
}

// Дочерний класс
public class Student : Person // Student наследует от Person
{
    public int StudentId { get; set; }
    public void Study()
    {
        Console.WriteLine($"Student {Name} is studying.");
    }
}

// Использование
Student student = new Student();
student.Name = "Alice"; // Свойство унаследовано от Person
student.Walk();         // Метод унаследован от Person
student.Study();        // Свой собственный метод
```
