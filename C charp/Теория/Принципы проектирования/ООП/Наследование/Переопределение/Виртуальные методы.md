*Виртуальный метод* - это метод в базовом классе, объявленный с модификатором `virtual`
Он предназначен для того, чтобы быть _переопределённым_ в производных классах

Виртуальные методы - основа *полиморфизма* (возможности объектов разных классов обрабатываться единым образом через общий интерфейс базового класса)
Они позволяют производным классам предоставлять свою собственную реализацию метода, которая будет вызываться даже если код работает с объектом через ссылку на базовый класс


**Отличия от обычного метода**
Виртуальный метод объявляется с модификатором `virtual`

Обычный метод обладает ранним связыванием и метод определяется на этапе компиляции по Типу переменной
Виртуальный метод обладает поздним связыванием и метод определяется во время выполнения кода, по типу фактически используемого объекта

Виртуальные методы немного медленнее из-за механизма виртуальных таблиц

Обычный метод нельзя переопределить, только скрыть
