*сокрытие (hiding)* - это когда в производном классе вы создаёте член (метод, свойство, поле) с таким же именем, как у базового класса, и помечаете его ключевым словом `new`

В результате:
- Член базового класса *не переопределяется*, а именно *прячется* за новым
- Поведение метода зависит от типа переменной, а не от типа объекта, который в этой переменной по факту хранится
```C#
class Base
{
    public void Show() => Console.WriteLine("Base.Show");
}

class Derived : Base
{
    public new void Show() => Console.WriteLine("Derived.Show");
}
```


Здесь `Derived.Show` *скрывает* `Base.Show`
Это **не** переопределение (`override`), потому что в базовом методе нет `virtual` и в производном нет `override`
```C#
Derived d = new Derived();
d.Show();      // Выведет: Derived.Show

Base b = d;
b.Show();      // Выведет: Base.Show
```
