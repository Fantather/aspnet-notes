Порядок вызова конструкторов строго определён: от базового класса к производному

1. При создании экземпляра дочернего класса сначала автоматически вызывается конструктор его *базового класса*
2. По умолчанию вызывается конструктор базового класса *без параметров*
3. Если базовый класс не имеет конструктора без параметров, дочерний класс _должен_ явно вызвать нужный конструктор базового класса с параметрами с помощью ключевого слова `base`
4. После завершения работы конструктора базового класса выполняется конструктор дочернего класса
