Свойство это метод-обёртка над полями, который выглядит и используется как поле, но позволяет добавлять собственную логику при получении или установке значения, возможностью ограничения доступа (private set) и проверки входящего значения

**Основная цель:** **Инкапсуляция**
Позволяет скрыть внутреннее состояние объекта (поле) и контролировать доступ к нему, избегая использования публичных полей

У `Set` есть скрытый параметр `value`, который хранит переданное значение

Автоматическое свойство - Инициализирует поле само
```C#
internal class Ex
{
	public int Properties { get; set; }
}
```

Свойство
```C#
private int number;
public int Properties
{
	get { return number; }
	set { number = value; }
}
```

1. Валидация при записи
   ```C#
   public int Properties
   {
	   set
	   {
		   if(value < 0) throw new ArgumentException();
		   age = value; // value - скрытый параметр Set
	   };
   }
```

2. Ограничение видимости
   ```C#
   public int Properties
   {
	   get; private set;
   }   
```

3. Init-only свойства
   Значение можно задать только в конструкторе или инициализаторе объекта
   ```C#
   public int Properties { get; init; }
```
