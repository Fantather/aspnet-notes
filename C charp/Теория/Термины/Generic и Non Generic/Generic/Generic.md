*Обобщённые типы* - Это классы, структуры, методы и интерфейсы, которые подстраиваются под нужный тип, при их использовании
Появились в C# 2.0

Все новые библиотеки, фреймворки и сам .NET ориентированы на [[Generic|generics]]
*Non-generic* коллекции (`ArrayList`, `Hashtable`) оставлены ради совместимости с очень старым кодом


**Преимущества generic коллекций**
- Позволяют заранее указать *тип данных*, с которым работает коллекция или класс
- Нет упаковки/распаковки для *value types*
- Строгая типизация: компилятор сразу запрещает несовместимые операции


Объявляются с параметрами типа (`<T>`)
```C#
public class Box<T> {
    public T Value { get; set; }
}

Box<int> intBox = new Box<int> { Value = 10 };
Box<string> strBox = new Box<string> { Value = "Hello" };
```
