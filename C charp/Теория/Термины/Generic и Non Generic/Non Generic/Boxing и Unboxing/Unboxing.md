*Распаковка* - это процесс извлечения значения *value type* из объекта, в котором оно было упаковано (*object*)
При этом значение из Кучи копируется обратно в Стек, а обёртка собирается [[GC]]

Это нужно делать явно - с помощью оператора приведения типов `(int)obj`

При этом [[CLR]] проверяет, что объект действительно содержит нужный *value type*
Если тип не совпадает, будет исключение `InvalidCastException`
```C#
object obj = 42;        // boxing
int x = (int)obj;       // unboxing
```
