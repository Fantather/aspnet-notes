*namespace* - это контейнер для идентификаторов (классов, функций, переменных и т. д.), который предотвращает конфликты имён и упорядочивает код

```C#
namespace Game {
    class Player { }
}

namespace Editor {
    class Player { }
}
```


*Назначение пространств имён*
Избегание конфликтов имён в крупных проектах, структурирование кода по смысловым областям, повышение читаемости и поддерживаемости


*Вложенные пространства имён используют*, когда нужно разделить код на более мелкие уровни внутри общей области и показать, как именно части проекта связаны между собой


*Разбиение пространства имён* на части позволяет размещать элементы одного пространства в разных файлах, при этом они образуют единое целое
