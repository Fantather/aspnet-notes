Преимущества
1. Копируется только ссылка, потому что это reference type
2. Плохо удаляется сборщиком мусора

---
Особенности
1. Можно создать объект с неинициализированными полями
2. Конструктор по умолчанию инициализирует ссылочные типы как null, числовые типы нулями
3. Конструктор с параметрами перекрывает Конструктор по умолчанию
4. Можно указать значение по умолчанию для полей, оно применится в классе не будет конструкторов
