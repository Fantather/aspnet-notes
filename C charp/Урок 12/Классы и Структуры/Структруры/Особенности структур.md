Особенности
1. Структуры копируются полностью
2. Нельзя создать объект с неинициализированными полями
3. Конструктор по умолчанию инициализирует ссылочные типы как null, числовые типы нулями
4. Конструктор с параметрами не перекрывает Конструктор по умолчанию
5. Нельзя указать значение по умолчанию для полей
