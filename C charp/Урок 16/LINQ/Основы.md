**Language Integrated Query (LINQ)** - это набор технологий, который позволяет работать с коллекциями с помощью удобных запросов, встроенных прямо в язык C#. Благодаря LINQ, можно писать запросы к данным так же, как обычный код, без отдельного SQL или парсинга

`LINQ` имеет очень высокий уровень абстракции, потому что они работают через `IEnumerable` поэтому один участок кода подходит под разные контейнеры
