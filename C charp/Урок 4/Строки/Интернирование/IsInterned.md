Метод `string.IsInterned` в C# используется для **проверки**, находится ли строка в **пуле интернированных строк**.

- Если переданная строка **уже интернирована** (то есть присутствует в пуле), метод возвращает **ссылку на эту строку из пула**.

- Если строка **не интернирована**, метод возвращает `null`.
