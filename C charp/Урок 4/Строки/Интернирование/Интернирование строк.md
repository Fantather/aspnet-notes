**Пул интернированных строк** (interned string pool) в C# — это **специальная область памяти**, в которой CLR (Common Language Runtime) хранит **уникальные экземпляры строковых литералов**, чтобы избежать дублирования одинаковых строк и экономить память.

Строковые литералы интернируются по умолчанию

```C#
string a = "hello";
string b = "hello";
```

Обе переменные `a` и `b` **ссылаются на один и тот же объект в памяти**, потому что `"hello"` — это строковый литерал, который **помещается в пул**.

___

**Если создать строку вручную** - Объект создаётся явно, это новый объект, не из пула
```C#
string a = "hello";
string b = new string("hello".ToCharArray());
Console.WriteLine(object.ReferenceEquals(a, b)); // false
```
