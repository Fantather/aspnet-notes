## `out` — "ссылка для возврата значения"

### Что делает:

Позволяет методу **вернуть значение через параметр**, не используя `return`.

### Правила:

- Переменную **не нужно инициализировать до передачи**.
    
- Метод **обязан присвоить ей значение** до выхода

```C#
void GetValues(out int a, out int b)
{
    a = 10;
    b = 20;
}

int x, y;
GetValues(out x, out y);
Console.WriteLine($"{x}, {y}"); // 10, 20

```
