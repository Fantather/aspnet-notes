## `ref` — "ссылка на уже инициализированную переменную"

### Что делает:

Позволяет методу **получить ссылку на уже существующую переменную** и **читать/изменять её значение**.

### Правила:

- Переменная **должна быть инициализирована до передачи** в метод.
    
- Внутри метода её **можно читать и писать**.


```C#
void Increment(ref int x) { x += 1; }  int a = 5; Increment(ref a); Console.WriteLine(a); // 6
```
