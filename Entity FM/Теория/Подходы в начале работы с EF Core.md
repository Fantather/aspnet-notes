*Code First* - рекомендуется для новых проектов, как более гибкий и современный подход
Мы начинаем с написания классов в C#(моделей)
EF Core создаёт и обновляет базу данных на основе этих классов с помощью миграций

*Database First* - для существующих Баз Данных
Мы начинаем с существующей базы данных
EF Core генерирует модели и контекст данных из схемы БД с помощью команды `Scaffold-DbContext`
