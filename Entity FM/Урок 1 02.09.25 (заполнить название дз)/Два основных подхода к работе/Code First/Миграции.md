Миграции в C# с EF Core — это способ **пошагового создания и изменения структуры базы данных** на основе твоих моделей (Code First). Они нужны, чтобы база синхронизировалась с кодом.

### Миграция — это класс с двумя методами:
```c#
protected override void Up(MigrationBuilder migrationBuilder)
{
    // инструкции для создания/изменения таблиц
}

protected override void Down(MigrationBuilder migrationBuilder)
{
    // инструкции для отката изменений
}

```
