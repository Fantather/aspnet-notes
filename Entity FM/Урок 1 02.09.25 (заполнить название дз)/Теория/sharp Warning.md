'#warning'
![[Pasted image 20250902190228.png]]

В C# `#warning` — это **директива препроцессора**, которая позволяет выводить предупреждение при компиляции.

```csharp
#warning Текст предупреждения
```

```csharp
#warning Надо переписать этот метод позже

public void OldMethod()
{
    // ...
}

```
- При компиляции в списке предупреждений появится текст: **«Надо переписать этот метод позже»**.
    
- Программа **соберётся**, но разработчик увидит предупреждение.
___
### Отличие от `#error`

- `#error` — выводит ошибку и **компиляция не пройдёт**.
```csharp
#error Этот метод нельзя использовать
```

Используется для:

- Напоминаний разработчику о недоделанных местах
    
- Временного отключения участков кода, которые нельзя использовать
    
- Контроля версий, платформ и условий компиляции
