## Определение типа через CLR

EF Core автоматически определяет тип столбца в базе данных на основе типа свойства в C# (CLR type):

|C# тип|SQL Server тип по умолчанию|Примечания|
|---|---|---|
|`int`|`int`|Числовой 32-битный|
|`long`|`bigint`|64-битный целочисленный|
|`short`|`smallint`|16-битный|
|`byte`|`tinyint`|0–255|
|`bool`|`bit`|0 или 1|
|`string`|`nvarchar(max)`|Можно ограничить длину через `[MaxLength]` или Fluent API|
|`DateTime`|`datetime2`|Можно указать точность через Fluent API|
|`decimal`|`decimal(18,2)`|Точность и масштаб можно настроить|
|`Guid`|`uniqueidentifier`|Для уникальных идентификаторов|
|`float`|`real`|32-битный|
|`double`|`float`|64-битный|
