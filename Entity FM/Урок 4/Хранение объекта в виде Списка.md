


**Настойка поведения Foreign Key при удалении**
- `CASCADE`  - при удалении строки из родительской таблицы автоматически удаляются все строки в дочерней таблице, которые на неё ссылаются.
- `SET NULL`  - в дочерней таблице в поле внешнего ключа будет установлено значение `NULL`, если родительская строка удалена.
- `RESTRICT` - удаление родительской строки запрещено, если на неё есть хотя бы одна ссылка из дочерней таблицы
