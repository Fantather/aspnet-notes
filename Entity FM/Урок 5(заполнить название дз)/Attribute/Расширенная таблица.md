[[Attribute]]

|Категория|Атрибут|Назначение|Пример|
|---|---|---|---|
|**EF Core – ключи и идентификаторы**|`[Key]`|Первичный ключ|`public int Id { get; set; }`|
||`[DatabaseGenerated(DatabaseGeneratedOption.Identity)]`|Автоинкремент|`public int Id { get; set; }`|
||`[DatabaseGenerated(DatabaseGeneratedOption.None)]`|Значение задаётся вручную|`public int Id { get; set; }`|
||`[DatabaseGenerated(DatabaseGeneratedOption.Computed)]`|Вычисляемое поле в БД|`public DateTime CreatedAt { get; set; }`|
|**Связи между сущностями**|`[ForeignKey("NavigationProperty")]`|Внешний ключ|`public int CategoryId { get; set; }`|
||`[InverseProperty("NavigationProperty")]`|Обратная навигация|`public ICollection<Order> Orders { get; set; }`|
||`[NotMapped]`|Поле игнорируется EF|`public int TempValue { get; set; }`|
|**Индексы и уникальность**|`[Index(IsUnique = true)]`|Уникальный индекс (EF Core 7+)|`public string Email { get; set; }`|
|**Concurrency и версии**|`[Timestamp]`|RowVersion для конкурентного доступа|`public byte[] RowVersion { get; set; }`|
||`[ConcurrencyCheck]`|Проверка конкурентного доступа|`public int Version { get; set; }`|
|**Точность чисел**|`[Precision(precision, scale)]`|Точность и масштаб числа|`[Precision(18,2)] public decimal Price { get; set; }`|
||`[Column(TypeName = "decimal(18,2)")]`|Альтернатива Precision|`public decimal Price { get; set; }`|
|**Строки и текст**|`[MaxLength(n)]`|Максимальная длина|`public string Name { get; set; }`|
||`[MinLength(n)]`|Минимальная длина|`public string Password { get; set; }`|
||`[StringLength(max, MinimumLength=min)]`|Длина строки|`public string Name { get; set; }`|
||`[Unicode(false)]`|Хранить как ASCII/не-Unicode|`public string Code { get; set; }`|
|**Даты и время**|`[DataType(DataType.Date)]`|Поле дата|`public DateTime BirthDate { get; set; }`|
||`[DataType(DataType.Time)]`|Поле время|`public TimeSpan StartTime { get; set; }`|
||`[DataType(DataType.Currency)]`|Валюта|`public decimal Salary { get; set; }`|
||`[DataType(DataType.MultilineText)]`|Многострочный текст|`public string Description { get; set; }`|
|**Валидация данных**|`[Required]`|Поле обязательно|`public string Name { get; set; }`|
||`[Range(min,max)]`|Диапазон значений|`public int Age { get; set; }`|
||`[RegularExpression("regex", ErrorMessage="")]`|Проверка формата строки|`public string Email { get; set; }`|
||`[EmailAddress]`|Проверка email|`public string Email { get; set; }`|
||`[Phone]`|Проверка телефона|`public string PhoneNumber { get; set; }`|
||`[Url]`|Проверка URL|`public string Website { get; set; }`|
||`[CreditCard]`|Проверка номера карты|`public string CardNumber { get; set; }`|
|**Сериализация**|`[Serializable]`|Позволяет сериализовать класс|`[Serializable] public class Person { ... }`|
||`[NonSerialized]`|Игнорировать поле при сериализации|`[NonSerialized] private int tempValue;`|
|**Устаревание**|`[Obsolete("сообщение", true/false)]`|Помечает устаревший метод/класс|`[Obsolete("Используй NewMethod", true)] public void OldMethod() {}`|
|**Shadow properties**|Атрибут не нужен|Поля есть в БД, но не в классе|EF Core создаёт скрытую колонку, доступ через `Entry()`|
|**Значения по умолчанию**|`[DefaultValue(...)]`|Значение по умолчанию в БД|`[DefaultValue(100)] public int Quantity { get; set; }`|
|**Порядок колонок**|`[Column(Order = n)]`|Определяет порядок столбцов в БД|`[Column(Order = 1)] public int Id { get; set; }`|
