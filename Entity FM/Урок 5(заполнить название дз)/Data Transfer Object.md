DTO —Data Transfer Object (Объект передачи данных).

По сути, это **специальный класс**, который используется для **переноса данных между слоями приложения** (например, между базой данных и UI), без лишней логики.

### Ключевые моменты:

- **Не содержит бизнес-логики** — только поля или свойства.
    
- Используется для **выборки только нужных данных**, чтобы не тянуть всю сущность из базы.
    
- Помогает **изолировать слои приложения** (например, Entity Framework + UI).
