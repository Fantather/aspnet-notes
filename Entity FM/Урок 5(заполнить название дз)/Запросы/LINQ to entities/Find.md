## **DbSet.Find() в EF Core**
[[Entity FM/Урок 5(заполнить название дз)/Круд операций#^first|First, First or Default]]
### 1. Определение

- Метод `Find` ищет объект в базе данных по **ключу (primary key)**.
    
- Работает **только по ключевым полям** сущности.
![[Pasted image 20250911202855.png]]

### 2. Особенности

- Сначала ищет в **контексте (кэше EF Core)**:  
    если объект уже загружен, запрос к БД не делается.
    
- Если объекта нет в контексте, отправляется SQL-запрос.
    
- Возвращает **`null`**, если объект не найден.

```c#
var product = context.Products.Find(5);

if (product != null)
{
    Console.WriteLine($"Найден продукт: {product.Name}");
}
else
{
    Console.WriteLine("Продукт не найден");
}

```


#### Для составного ключа:

```c#
var entity = context.Entities.Find(key1, key2);
```

### 5. Отличие от `FirstOrDefault`

| Метод            | Когда использовать         |
| ---------------- | -------------------------- |
| `Find`           | Быстрый поиск по **ключу** |
| `FirstOrDefault` | Любые условия (`Where`)    |
