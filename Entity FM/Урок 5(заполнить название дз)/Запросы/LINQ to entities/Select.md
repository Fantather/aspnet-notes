Все телефоны коллекции на анонимный объект

Когда мы более узко умещаем объект когда нам надо только пара объектом

мапим селектом на новый анонимный объект

```c#
var result = context.Table
    .Where(x => /* условие */)
    .Select(x => new 
    {
        Поле1 = x.Column1,
        Поле2 = x.Column2,
        Вычисляемое = x.Column3 * 0.9
    })
    .ToList();

```
**Особенности:**

- `new { ... }` создаёт **анонимный объект** с выбранными полями.
    
- Можно добавлять вычисляемые поля прямо в `Select`.
    
- EF Core переводит это в SQL, выбирая только нужные столбцы.
    
- Если нужен **[[Data Transfer Object|DTO]]**, заменяем анонимный объект на класс:
```c#
.Select(x => new MyDto {
    Field1 = x.Column1,
    Field2 = x.Column2
})

```
