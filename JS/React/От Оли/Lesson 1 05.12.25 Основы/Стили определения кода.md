### Императивный стиль (Imperative)

**Суть:** Вы описываете **КАК** компьютер должен решить задачу. Это пошаговая инструкция. Вы управляете потоком выполнения (циклы, условия) и состоянием переменных вручную.

- **Принцип:** "Сделай шаг А, затем шаг Б, если условие В верно — сделай шаг Г".
    
- **Аналогия:** Кулинарный рецепт (Возьми яйцо, разбей, вылей на сковороду, жарь 5 минут).
    
- **Примеры:** C, C++, старый Pascal, классическое использование циклов `for` и `while`.
    

#### Подтипы императивного стиля:

1. **Структурное программирование:** Код разбит на блоки (функции/процедуры), используются `if`, `while`.
    
2. **Объектно-ориентированное (ООП):** Состояние программы и логика упакованы в объекты (Классы). C# и Java — яркие представители (хотя они умеют и в декларативный стиль).

### Декларативный стиль (Declarative)

**Суть:** Вы описываете **ЧТО** вы хотите получить в итоге. Вы не говорите, _как_ именно это сделать — детали реализации скрыты "под капотом" языка или библиотеки.

- **Принцип:** "Мне нужен список пользователей старше 18 лет".
    
- **Аналогия:** Заказ в ресторане (Принесите мне стейк. Вам не важно, как именно повар его готовит).
    
- **Примеры:** SQL (базы данных), HTML/CSS (вы не пишете алгоритм отрисовки кнопки, вы просто пишете `<button>`), Регулярные выражения.
    

#### Подтип: Функциональное программирование (ФП)

Популярный сейчас стиль (активно используется в React, современном JS и LINQ в C#). Здесь программы строятся из чистых функций, без изменения переменных.

### Наглядный пример (JavaScript)

Задача: У нас есть массив чисел `[1, 2, 3]`. Нужно умножить каждое число на 2.

#### Императивный подход (Цикл `for`)

Мы говорим машине: создай счетчик `i`, увеличивай его, бери элемент по индексу, умножай, клади в новый массив.



```JavaScript
const numbers = [1, 2, 3];
const doubled = [];

for (let i = 0; i < numbers.length; i++) {
    doubled.push(numbers[i] * 2);
}

console.log(doubled); // [2, 4, 6]
```

#### Декларативный подход (Метод `map`)

Мы говорим машине: "Отобрази (map) этот массив так, чтобы каждый элемент умножился на 2". Нам всё равно, как там внутри крутится цикл.



```JavaScript
const numbers = [1, 2, 3];
const doubled = numbers.map(n => n * 2);

console.log(doubled); // [2, 4, 6]
```
