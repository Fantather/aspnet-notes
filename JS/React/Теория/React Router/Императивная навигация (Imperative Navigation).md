#### Что это такое?

**Императивная навигация** — это способ перемещения пользователя по приложению с помощью **JavaScript-кода**, а не через клик по ссылке (`<Link>`).

Это нужно, когда переход должен произойти автоматически после какого-то события:
- Успешная отправка формы.
- Завершение таймера.
- Выход из системы (Logout).

#### Основной инструмент: `useNavigate`

В React Router v6+ для этого используется хук `useNavigate`. Он возвращает функцию, которая позволяет управлять историей браузера.

**Важно:** Сам хук всегда импортируется как `useNavigate`, но переменную, в которую мы сохраняем результат его работы, мы можем назвать как угодно (в нашем случае — `Maps`).

#### Инициализация
```JSX
import { useNavigate } from 'react-router-dom';

function MyComponent() {
  // Получаем функцию навигации и называем её Maps
  const Maps = useNavigate(); 
  
  // ...остальной код
}
```

#### Основные сценарии использования

**Переход на конкретный путь**
Передает пользователя на указанный URL
По умолчанию это добавляет новую запись в историю браузера (кнопка "Назад" вернет пользователя на предыдущую страницу)
```JSX
const handleSubmit = () => {
  // Логика сохранения данных...
  
  // Перенаправляем на главную
  Maps('/'); 
  
  // Или на страницу профиля
  Maps('/profile');
};
```

**Навигация по истории (Delta Navigation)**
Мы можем передать число, чтобы переместиться назад или вперед по истории посещений
```JSX
// Вернуться на 1 страницу назад (аналог кнопки браузера "Назад")
<button onClick={() => Maps(-1)}>
  Назад
</button>

// Вернуться на 2 страницы назад
Maps(-2);

// Вперед на 1 страницу (аналог кнопки браузера "Вперед")
Maps(1);
```

**Замена текущей записи (Replace)**

Иногда нам нужно заменить текущую страницу в истории, а не добавить новую. Это критично для страниц авторизации: чтобы после входа пользователь, нажав "Назад", не попал снова на форму логина.

Для этого вторым аргументом передается объект опций:
```JSX
const handleLogin = () => {
  // Пользователь вошел, перекидываем в дашборд
  // replace: true стирает страницу логина из истории переходов
  Maps('/dashboard', { replace: true });
};
```

#### Сводная таблица аргументов `Maps()`

|**Аргумент**|**Тип**|**Описание**|**Пример**|
|---|---|---|---|
|`to`|`string`|Путь, куда нужно перейти.|`Maps('/cart')`|
|`delta`|`number`|Смещение в истории (назад/вперед).|`Maps(-1)`|
|`options`|`object`|Настройки перехода (например, `replace`).|`Maps('/', { replace: true })`|
