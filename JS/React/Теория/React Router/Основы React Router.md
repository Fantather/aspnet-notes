Это стандартная библиотека для маршрутизации в экосистеме React, которая позволяет создавать **Single Page Applications (SPA)**.

Основная задача React Router — синхронизировать UI (интерфейс) с URL-адресом браузера. В отличие от традиционного серверного роутинга, где при переходе по ссылке браузер запрашивает новый HTML-документ с сервера, React Router перехватывает изменение URL, предотвращает перезагрузку страницы и манипулирует DOM, отображая нужные компоненты.

Технически библиотека опирается на **History API** браузера, что позволяет управлять историей сеанса (переходы назад/вперед) и состоянием приложения без лишних сетевых запросов


### Установка
В экосистеме React для веба мы используем пакет `react-router-dom`
То есть команда `npm i react-router-dom`
