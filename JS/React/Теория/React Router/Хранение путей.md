Обычно создается файл `routes.js` или `paths.js`:
```JSX
export const ROUTE_PATHS = {
  HOME: '/',
  REGISTER_STEP_1: '/register/step-1',
  REGISTER_STEP_2: '/register/step-2',
  REGISTER_STEP_3: '/register/step-3',
  SUMMARY: '/register/summary',
};
```

И затем используется в компонентах:
```JSX
// Импорт константы вместо написания строки вручную
<Link to={ROUTE_PATHS.REGISTER_STEP_1}>Начать регистрацию</Link>
```


Так же в файл рекомендую добавить метод `getRoutePaths()` или что-то такое, который бы возвращал наш объект в виде массива
```JS
export function getRoutePaths()
{
	return Object.values(ROUTE_PATHS);
}
```
