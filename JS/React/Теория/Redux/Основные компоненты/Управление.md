Источник состояния (store) - единственный источник истинны
Позволяет получать доступ, что бы подписываться на изменения

Действия (actions) - Простой JS объект, который описывает что произошло, способ доставить данные в хранилища

reducers - функции, которые создают действия
Он не изменяет данные, он создаёт копию с изменениями
