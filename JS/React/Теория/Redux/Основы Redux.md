Библиотека для управления глобальным состоянием приложения
Она помогает работать с состоянием проще, более централизованно, более оптимизированно и быстро

Написан на чистом js и используется во всех JS-фреймворках

Состоит из двух слов reduce и flux (архитектура, когда данные движутся только в одну сторону)
