`src/assets` - для статических ресурсов приложения
В этой папке хранятся файлы, которые импортируются непосредственно в компоненты JavaScript
Это могут быть изображения (png, jpg, svg), шрифты или глобальные таблицы стилей

В отличие от папки `public`, файлы из `assets` обрабатываются сборщиком (Vite/Webpack) при сборке проекта: они могут быть оптимизированы, переименованы (хешированы) или преобразованы в base64-строки.
