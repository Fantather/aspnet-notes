`src/components` - для наших [[Компоненты|Компонентов]]

Такие компоненты часто содержат пере используемые части интерфейса, название компонента всегда должно соответствовать названию файла

Так же, если этот компонент использует стили, которые нужны только ему, то мы создаём для него css-файл с таким же названием

Если компонентов много, то абсолютно нормально группировать их по папкам внутри `components`
На пример `components/admin-panel` `components/user-panel`
