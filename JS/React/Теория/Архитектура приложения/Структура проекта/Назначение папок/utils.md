`src/utils` - утилитарные функции, они не содержат никакого JSX кода, это обычные JS функции, которые занимаются любой вспомогательной работой

На пример форматирование данных, генерация случайных значений, генерация email, проверка валидности данных, работа со строками, преобразование массивов, логические проверки
