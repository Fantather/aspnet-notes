### App.jsx
`src/App.jsx` - корневой [[Компоненты|Компонент]] приложения

Это главный компонент, который объединяет все остальные части интерфейса
Обычно именно здесь настраивается общая структура страницы (Layout) или определяются маршруты (Routing) приложения
Все остальные компоненты вкладываются внутрь `App`

### App.css
`src/App.css` - файл стилей для корневого компонента `App.jsx`

Обычно здесь содержатся CSS-правила, которые относятся непосредственно к разметке, определенной в компоненте `App`
Если используется модульный подход, файл может называться `App.module.css` для изоляции стилей
