`eslint.config.js` - конфигурация линтера ESLint
Файл настроек инструмента статического анализа кода. Он определяет набор правил, которым должен соответствовать написанный код

ESLint помогает:
- Выявлять синтаксические ошибки до запуска кода
- Находить потенциальные баги (например, использование необъявленных переменных).
- Поддерживать единый стиль кодирования (Code Style) во всей команде
