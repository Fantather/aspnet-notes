`.gitignore` - конфигурация исключений для Git

Текстовый файл, указывающий системе контроля версий Git, какие файлы и директории следует игнорировать

Это критически важно для исключения из репозитория:
- Папки `node_modules` (зависимости должны устанавливаться локально, а не храниться в Git)
- Папки сборки (`dist`, `build`)
- Файлов с локальными настройками среды (`.env.local`), которые могут содержать приватные ключи
- Системных файлов ОС (например, `.DS_Store`)
