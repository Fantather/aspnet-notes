### index.css

`src/index.css` - глобальные стили проекта

В этом файле прописываются стили, которые применяются ко всему приложению целиком. Здесь обычно находятся:
- Сброс стилей браузера (CSS Reset или Normalize)
  
- Определение CSS-переменных (цветовая палитра, шрифты)
  
- Базовые настройки типографики (`font-family`, `font-size`) для тега `body`
  
### index.html

`index.html` - входная точка HTML

Единственный HTML-файл в Single Page Application (SPA). Он содержит:
- Мета-теги (viewport, title, description
  
- Подключение иконок (favicon)
  
- Пустой контейнер `<div id="root"></div>`, куда React монтирует все приложение
  
- Скрипт подключения `main.jsx` (в случае использования Vite, он подключается как модуль `<script type="module">`)
