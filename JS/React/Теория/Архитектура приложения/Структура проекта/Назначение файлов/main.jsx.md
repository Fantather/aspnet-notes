`src/main.jsx` - точка входа (Entry Point) JavaScript
Это скрипт, с которого начинается выполнение React-приложения

Его основные задачи:
1. Найти HTML-элемент (обычно с `id="root"`) в DOM-дереве
2. Инициализировать React DOM
3. Отрендерить компонент `<App />` внутрь найденного элемента
4. Подключить глобальные провайдеры (например, `BrowserRouter`, `Redux Provider` или `StrictMode`)

Мы в нём ничего не пишем, все изменения и рендер передаём в App
