`package-lock.json` - фиксация дерева зависимостей
Автоматически генерируемый файл, который описывает точное дерево зависимостей, установленных в данный момент

Он фиксирует конкретные версии всех пакетов и их под-зависимостей. Это гарантирует, что у всех разработчиков в команде и на сервере (CI/CD) будет установлен абсолютно идентичный набор пакетов ("Works on my machine" проблема исключается)
