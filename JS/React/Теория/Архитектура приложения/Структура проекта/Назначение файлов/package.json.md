`package.json` - манифест проекта
Главный конфигурационный файл Node.js проекта в формате JSON

Он содержит:
- Метаданные проекта (имя, версия, автор, лицензия)
- `scripts` — набор команд для запуска, сборки, тестирования и линтинга проекта
- `dependencies` — список библиотек, необходимых для работы приложения в продакшене (например, `react`)
- `devDependencies` — список инструментов, нужных только для разработки (например, `vite`, `eslint`)
