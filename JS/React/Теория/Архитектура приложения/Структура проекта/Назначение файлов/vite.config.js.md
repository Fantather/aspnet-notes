`vite.config.js` - конфигурация сборщика Vite
Файл, экспортирующий настройки для инструмента сборки Vite

Здесь определяются:
- Плагины (например, `@vitejs/plugin-react` для поддержки React и JSX)
- Настройки локального сервера (порт, проксирование запросов к API)
- Алиасы путей (alias) для упрощения импортов (чтобы писать `@/components` вместо `../../components`)
- Параметры оптимизации сборки
