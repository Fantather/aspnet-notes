Нужно для CSS, для чего-то с overflow окошками (Это компоненты в которых есть скролл)

Для модальных окон, если родительский компонент имеет низкий z-индекс, модальное окно тоже будет его иметь, но мы можем вызвать это модальное окно в коревом компоненте через порталы

Семантическая чистота - Модальные окна, глобальные уведомления не должны принадлежать какому-то конкретному объекту или кнопке
Они принадлежат всему приложению
