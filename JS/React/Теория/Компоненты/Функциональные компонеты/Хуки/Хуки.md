Есть множество хуков и мы можем определять свои
Это самые обычные функции, цель которых - облегчить нам жизнь
Хуки работают только в функциональных компонентах

Не нужно объявлять хуки просто так, только по мере необходимости для улучшения читаемости кода

Хуки можно вызывать только на верхних уровнях, нельзя вызывать в циклах, ......
Нельзя вызывать хуки из обычного JS, исключение это наши собственные хуки


Пользовательские хуки должны
Хранится в папке *hooks*
Называться с префикса `use`, обязательно с маленькой буквы
Использоваться только в функциональных компонентах или пользовательских хуках
