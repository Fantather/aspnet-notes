Каждый cookie состоит из имени, значения и набора атрибутов, определяющих его поведение

| **Атрибут**    | **Описание и техническое поведение**                                                                                                                                                                                                                                                                                                                                                                                                       |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Name=Value** | Обязательная часть. Допускаются любые US-ASCII символы, кроме управляющих и разделителей (пробелы, табы, `;`). Рекомендуется использовать URL-кодирование (`encodeURIComponent`).                                                                                                                                                                                                                                                          |
| **Expires**    | Устанавливает абсолютную дату/время истечения срока действия (формат HTTP-date). Если не указано, cookie является **сессионным** (удаляется при закрытии браузера).                                                                                                                                                                                                                                                                        |
| **Max-Age**    | Альтернатива `Expires`. Устанавливает время жизни в секундах (относительно текущего момента). Имеет приоритет над `Expires`.<br><br>`Max-Age=0` или отрицательное число удаляет cookie.                                                                                                                                                                                                                                                    |
| **Domain**     | Определяет хосты, на которые будет отправляться cookie.<br><br><br>Если не указан: только текущий хост (без поддоменов).<br><br>Если указан (например, `example.com`): включает сам домен и все его поддомены (`api.example.com`, `www.example.com`).                                                                                                                                                                                      |
| **Path**       | Ограничивает область действия cookie определенным путем в URL. Например, `Path=/docs` означает, что cookie будет отправлен только для запросов, начинающихся с `/docs`. Стандартное значение: `/`.                                                                                                                                                                                                                                         |
| **Secure**     | Директива, указывающая браузеру отправлять cookie **только** по зашифрованному протоколу HTTPS. На `localhost` (исключение) может работать по HTTP.                                                                                                                                                                                                                                                                                        |
| **HttpOnly**   | **Критически важный атрибут.** Запрещает доступ к cookie через JavaScript API (`document.cookie`). Защищает от кражи токенов при XSS-атаках.                                                                                                                                                                                                                                                                                               |
| **SameSite**   | Контролирует отправку cookie при кросс-сайтовых запросах (Cross-Site requests) для защиты от CSRF.<br><br><br>• `Strict`: Cookie отправляется только в First-Party контексте (когда сайт в адресной строке совпадает с доменом cookie).<br><br>• `Lax` (default): Отправляется при навигации по ссылкам (GET), но не при фоновых запросах (AJAX, Image) с других сайтов.<br><br>• `None`: Отправляется всегда (требует атрибута `Secure`). |
