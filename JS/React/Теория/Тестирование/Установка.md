## Установка
Установка Vitest
`npm install -D vitest`
Вписываем `"test": "vitest"` в раздел scripts в файл "package.json"

Установка Jest
`npm install --save-dev jest`
Вписываем `"test": "jest"` в раздел scripts в файл "package.json"


## Именование и расположение
Тестовые файлы должны находиться в папке __tests__ или рядом с компонентом или, что более популярно, заканчиваться на .test.js или .spec.js


## Запуск теста
Написать в консоли `npm test`
