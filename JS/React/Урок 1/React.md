Это открытая JS библиотека, разработанная Facebook для создания пользовательских интерфейсов Веб-приложений
Позволяет строить разработчикам многокомпонентные, интерактивные и масштабируемые пользовательские интерфейсы с помощью компонентного подхода

Изначально создавался для создания Веб-приложений(Веб-сайтов)
Позже была создана платформа React Native для мобильных устройств и сейчас *React* представляет собой идеальный инструмент для создания масштабируемых SPA Веб-Приложений (Одностраничных сайтов)

*React* работает с [[Виртуальный DOM|Виртуальным DOM]], а не обычным
Если приложению необходимо узнать информацию о состоянии элементов, то происходит обращение именно к *Виртуальному DOM*
Если необходимо изменить элемент Веб страницы, то изменения сначала вносятся в виртуальный [[DOM|DOM]], потом состояние виртуального *DOM* сравнивается с состоянием реального *DOM*, если эти состояния отличаются, то *React* за нас находит минимально необходимое количество манипуляций, для синхронизации реального *DOM* и виртуального и выполняет их
Такая схема для взаимодействия с элементами веб-страницы работает гораздо быстрее и эффективнее, чем работа через чистый [[Основная информация о Java Script|JavaSctipt]]

**Особенности React**
- *Компоненты*
  Блоки которые можно повторно использовать

- *JSX* - комбинация кода *JS* и *XML*, расширение JS-кода
  Простой и понятный способ для определения кода вашего визуального интерфейса
  В *JS-коде* мы будем писать код похожий на *HTML*, а он будет транслироваться в *JS-вызов*
