Рендеринг элемента на веб-странице разбивается на два этапа

Вначале нам надо установить элемент веб-страницы, в который будет производиться рендеринг приложения React. Для этого применяется метод `ReactDOM.createRoot()`

В него передается элемент веб-страницы, в котором будет производиться рендеринг
В данном случае это элемент `<div id="app"></div>` 

```HTML
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>ItStep</title>
</head>
<body>
    <div id="app"> </div>
    
    <!-- crossorigin для работы с кроссбраузерными запросами -->
    <script crossorigin src="https://unpkg.com/react@18/umd/react.production.min.js"></script>
    <script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"></script>

    <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
    <script type="text/babel">
        const rootNode = document.getElementById("app");
        const root = ReactDOM.createRoot(rootNode);
        root.render(
            <h1>Hello React</h1>
        );
    </script>
</body>
</html>
```

`<div id="app"> </div>` - служит контейнером, куда *React* будет вставлять всё приложение, единственный HTML элемент в коде
``
`<script crossorigin src="https://unpkg.com/react@18/umd/react.production.min.js">` и
`<script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js">` - библиотеки необходимые для работы *React*

`<script type="text/babel">` - атрибут `type="text/babel"` указывает, что данный код должен обрабатываться выше подключённым компилятором *Babel*

`const root = ReactDOM.createRoot(rootNode);` - Превращаем HTML-элемент в React-корень

`root.render` - Рисуем содержимое скобок внутри корня
