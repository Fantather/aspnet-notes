Библиотека это набор функций и классов, которые используются для решения конкретных задач
Программа может использовать множество библиотек, каждая библиотека решает свою конкретную задачу

Фреймворк это основа, на которой строится приложение, которая определяет его архитектуру и структуру
Фреймворк контролирует абсолютно всё, разработчик добавляет свой код в определённые места и фреймворк вызывает этот код по мере необходимости

Фреймворки обычно решают более широкие задачи, чем библиотеки, поскольку они предоставляют инструменты и структуры для построения целых приложений
