```JSX
import React from 'react'

function App() {
  return (
    <div>
      <h1>
        First Lesson
      </h1>
    <MyComponent name = "Funckyy"/>

    <MyClassComponent name = "Classy"/>
    </div>
  )
}

export default App
function MyComponent(props)
{
  return <div>I am func component! My name is - {props.name}</div>;
}

class MyClassComponent extends React.Component {
  render() {
    return <div>I am class component! My name is - {this.props.name}</div>;
  }
}
```

`props` - свойство доступное только для чтения



```JSX
function MyComponent({name})
{
  return <div>I am func component! My name is - {name}</div>;
}
```

### Значения по умолчанию
**Функциональные компоненты**
```JSX
function MyComponent({name, job = "Component"})
{
  return <div>I am func component! My name is - {name}, job - {job}</div>;
}
```

**Классовые компоненты**
Может выглядеть дико, но если компонентов по умолчанию много, этот способ может быть удобным
```JSX
class MyClassComponent extends React.Component {
  render() {
    return <div>I am class component! My name is - {this.props.name}, job - {this.props.job}</div>;
  }
}

MyClassComponent.defaultProps = {job: "Component"}
```
