JSX синтаксис
Он максимально облегчит работу и уменьшит объёмы текста
```JSX
class MyComponent extends React.Component {
  render() {
    return 'I am class';
  }
}
```


JS синтаксис
О нём нужно хотя бы знать, хотя бы потому что JSX-синтаксис это 
```JS
class MyComponent extends React.Component {
  render() {
    return React.createElement("span", null, "I am class");
  }
}
```
