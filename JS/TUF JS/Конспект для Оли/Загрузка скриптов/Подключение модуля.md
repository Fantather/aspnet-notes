`<script type="module">`
Модули дают возможность загружать файлы скриптов по мере необходимости через `import`
Скрипт будет оттуда подгружаться в оперативную память только во время вызова
![[Pasted image 20251128190001.png]]


Так же `type="module"` даёт возможность писать `await` где угодно внутри скрипта
Но лучше всё-таки использовать `await` внутри async-функции
