Если браузер встречает при выполнении HTML страницы  подключение ещё какого-либо файла(JS, CSS, картинка), то Браузер обращается на сервер ещё и за этим подключаемым файлом

Сервер возвращает специальный HTTP-ответ с полями `Cache-Control` и `ContentType`

Поле `ContentType` - которое обозначает тип передаваемого файла
Этот файл браузер добавит себе в кеш

Поле `max-age` в `Cache-Control` хранит в себе время, которое будет храниться этот кеш в секундах
31536000 - один год
