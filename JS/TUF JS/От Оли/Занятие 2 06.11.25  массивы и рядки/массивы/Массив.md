## Массивы в JavaScript (Array)

**Массив (Array)** — это специальный тип объекта, предназначенный для хранения **упорядоченной коллекции** элементов. Массивы позволяют хранить несколько значений в одной переменной.

---

### 1. Создание массива

**A. Литеральный синтаксис (предпочтительный)** Это самый простой, распространенный и "похожий на Python" способ. 
![[Pasted image 20251106201758.png]]



```JavaScript
// Заполнение при создании
let fruits = ["Яблоко", "Апельсин", "Слива"];
let clubs = ["Динамо", "Шахтер", "Черноморец"];

// Пустой массив
let emptyArray = [];
```

**Б. Через конструктор `new Array()` (старый способ)** 
![[Pasted image 20251106201730.png]]


```JavaScript
let clubs = new Array("Динамо", "Шахтер"); // То же, что ["Динамо", "Шахтер"]
let emptyArray2 = new Array(); // То же, что []
```

---

### 2. Ключевые особенности

- **Нумерация с нуля (Zero-indexed):** Как вы и отметили, доступ к элементам массива всегда начинается с индекса **0**.
    
    - `clubs[0]` — это "Динамо"
        
    - `clubs[1]` — это "Шахтер"
        
- **Смешанные типы:** Как вы и указали ("МЕСТИТЬ В СЕБЕ ВСЕ ПОДРЯД"), массив в JavaScript может хранить элементы любых типов одновременно. `

![[Pasted image 20251106195856.png]]   


```JavaScript
let mixed = [1, "Привет", true, { name: "Alex" }, [10, 20]];
```


- **Разреженные массивы (Sparse Arrays):** Это ваше наблюдение о "100-м элементе". Массивы в JavaScript могут быть "разреженными". Если у вас есть массив из 3-х элементов и вы присваиваете значение 100-му, массив автоматически увеличит свою длину. 

![[Pasted image 20251106202101.png]]

```JavaScript
    let clubs = ["Динамо", "Шахтер", "Черноморец"]; // length: 3
    
    // Добавляем элемент далеко за пределами
    clubs[5] = "Днепр";
    
    console.log(clubs.length); // Выведет: 6
    console.log(clubs);
    // Вывод в консоли:
    // ["Динамо", "Шахтер", "Черноморец", undefined, undefined, "Днепр"]
    // (на самом деле `empty x 2`, но при доступе они ведут себя как undefined)
    ```

### 3. Доступ к элементам

- **Доступ по индексу `[]`:**

```JavaScript
    let clubs = ["Динамо", "Шахтер"];
    console.log(clubs[0]); // "Динамо"
    ```

- **Доступ за пределами (Ваш пример):** Если вы обращаетесь к индексу, которого не существует, вы **не получите ошибку**. Вы просто получите значение `undefined`. 
 ![[Pasted image 20251106201949.png]]
    
    

```JavaScript
console.log(clubs[5]); // Выведет: undefined
```

### 4. Перебор массива (Итерация)

Есть несколько способов "пробежать" по массиву.

**A. Классический цикл `for`** Используется, когда вам нужен доступ к индексу (`i`).
```JavaScript
let clubs = ["Динамо", "Шахтер", "Черноморец"];
for (let i = 0; i < clubs.length; i++) {
  document.write(clubs[i] + "<br />");
}
```

**Б. Цикл `for...in` (Перебор индексов)** Как вы и отметили (`for (var something in somth)`), этот цикл перебирает **индексы** (или "ключи"). Он **не рекомендуется** для массивов, но важно знать, как он работает. 
![[Pasted image 20251106201907.png]]

```JavaScript
for (let index in clubs) {
  // 'index' будет "0", "1", "2" (причем это будут строки, а не числа)
  document.write(index + ": " + clubs[index] + "<br />"); 
}
```

**В. Цикл `for...of` (Современный способ, перебор значений)** Этот цикл (введен в ES6) перебирает **значения** массива. Это самый простой и чистый способ, если вам не нужен индекс.

```JavaScript
for (let club of clubs) {
  // 'club' будет "Динамо", "Шахтер", "Черноморец"
  document.write(club + "<br />");
}
```

### 5. Массивы и строки

- **Прямой вывод (Ваш пример):** Когда вы пытаетесь вывести массив как строку (например, `document.write(clubs)`), JavaScript автоматически объединяет все элементы в одну строку, разделяя их **запятой**. `![[Pasted image 20251106201758.png]]`

```JavaScript
    let clubs = ["Динамо", "Шахтер"];
    document.write(clubs); // Выведет на экран: Динамо,Шахтер
```
