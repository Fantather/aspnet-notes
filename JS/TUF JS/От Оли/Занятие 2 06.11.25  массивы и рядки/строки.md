## Строки в JavaScript (String)

**Строка (String)** — это примитивный тип данных в JavaScript, предназначенный для хранения последовательности символов (текстовой информации).

Ключевая особенность строк в JavaScript — они **неизменяемые (immutable)**.

---

### 1. Создание строки

Строку можно создать тремя способами (используя разные кавычки):

1. **Двойные кавычки:** `let str = "Привет";`
    
2. **Одинарные кавычки:** `let str = 'Привет';`
    
3. **Обратные кавычки (Шаблонные литералы):** `let str = \`Привет`;`
    

**Шаблонные литералы (`...`)** Это современный (ES6+) и самый гибкий способ. Он позволяет:

- **Встраивать выражения (Интерполяция):** Вставлять переменные и выражения прямо в строку с помощью `${...}`.
    
- **Создавать многострочный текст:** Просто перенося строку в редакторе.
    



```JavaScript
let name = "Alex";
let price = 100;

// Интерполяция
let greeting = `Привет, ${name}!`; // Результат: "Привет, Alex!"

// Многострочность
let text = `
Строка 1
Строка 2
`;
```

---

### 2. Ключевая особенность: Неизменяемость (Immutability)

Это **самое важное** свойство строк. Нельзя изменить символ _внутри_ уже существующей строки. Любой метод, который "меняет" строку, на самом деле **возвращает новую строку**.



```JavaScript
let str = "Привет";

// Попытка изменить символ
str[0] = "п"; // ОШИБКИ НЕ БУДЕТ, но...

console.log(str); // Выведет "Привет", а не "привет"
```

Чтобы "изменить" строку, нужно создать новую, например, путем конкатенации (сложения):


```JavaScript
let str = "Привет";
str = str + ", мир!"; // str теперь ссылается на НОВУЮ строку "Привет, мир!"
```

---

### 3. Доступ к символам

Получить конкретный символ из строки можно двумя способами:

1. **Квадратные скобки `[]` (Современный способ)**

 ```JavaScript
    let str = "Мир";
    console.log(str[0]); // "М"
    
    // Если индекса нет, вернет undefined
    console.log(str[99]); // undefined
    ```
    
2. **Метод `charAt(index)` (Старый способ)**
 
  ```JavaScript
    let str = "Мир";
    console.log(str.charAt(0)); // "М"
    
    // Если индекса нет, вернет пустую строку
    console.log(str.charAt(99)); // ""
    ```
    

---

### 4. Основные свойства и методы

_(Помните: все методы, которые "меняют" строку, возвращают новую, не трогая старую)._

#### Свойство `length`

- **`length`**: Возвращает длину строки (количество символов).

 ```JavaScript
    "JavaScript".length; // 10
    ```
    

#### Поиск подстроки

- **`includes(substr)`**: Проверяет, **содержит** ли строка `substr`. Возвращает `true` или `false`.
 

  ```JavaScript
    "Привет, мир".includes("мир"); // true
    ```
    
- **`indexOf(substr, [pos])`**: Ищет `substr`, начиная с позиции `pos` (необязательно). Возвращает **индекс** первого вхождения или **-1**, если ничего не найдено.
 
 ```JavaScript
    "JavaScript".indexOf("Script"); // 4
    "JavaScript".indexOf("CSS"); // -1
    ```

- **`startsWith(substr)` / `endsWith(substr)`**: Проверяет, начинается ли (`startsWith`) или заканчивается ли (`endsWith`) строка на `substr`. Возвращает `true` / `false`.


#### Извлечение части строки

- **`slice(start, [end])`**: **(Самый используемый)** Возвращает **новую строку** — "срез" от `start` до `end` (не включая `end`).
    
    - `start` и `end` могут быть отрицательными (означают отсчет с конца).
    - 
 ```JavaScript
    let str = "JavaScript";
    
    str.slice(0, 4);  // "Java" (с 0 по 4)
    str.slice(4);     // "Script" (с 4 до конца)
    str.slice(-3);    // "ipt" (последние 3 символа)
    ```

- **`substring(start, [end])`**: Похож на `slice`, но не поддерживает отрицательные аргументы и меняет местами `start` и `end`, если `start > end`.
    

#### Изменение регистра

- **`toLowerCase()`**: Приводит строку к нижнему регистру.
    
- **`toUpperCase()`**: Приводит строку к верхнему регистру.
    
    JavaScript
    
```JavaScript
    "JavaScript".toLowerCase(); // "javascript"
    ```


#### Прочие полезные методы

- **`trim()`**: Убирает пробелы (и другие невидимые символы) в **начале и конце** строки.

  ```JavaScript
   "   Привет   ".trim(); // "Привет"
    ```
    
- **`split(separator)`**: **(Очень важный)** "Разбивает" строку на **массив** по указанному разделителю `separator`.


 ```JavaScript
    // 1. Разделение по запятой
    let csv = "Яблоко,Апельсин,Груша";
    let fruits = csv.split(',');
    // fruits: ["Яблоко", "Апельсин", "Груша"] (это уже массив)
    
    // 2. Разделение на буквы (пустой разделитель)
    let word = "Привет";
    let letters = word.split('');
    // letters: ["П", "р", "и", "в", "е", "т"]
    ```
    
- **`replace(search, replace)`**: Ищет `search` (строку или регулярное выражение) и заменяет **первое вхождение** на `replace`.


  ```JavaScript
    "JavaScript это круто".replace("JavaScript", "JS");
    // "JS это круто"
    ```
    
- **`replaceAll(search, replace)`**: Заменяет **все** вхождения.
