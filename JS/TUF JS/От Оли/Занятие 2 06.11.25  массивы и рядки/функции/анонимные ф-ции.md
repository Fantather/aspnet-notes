## Анонимные функции и Функциональные выражения (Function Expression)

### 1. Анонимная функция

**Анонимная функция** — это функция, у которой нет собственного имени (идентификатора) после ключевого слова `function`.

**Пример (сама анонимная функция):**

```JavaScript
function() {
  document.write("Анонимная функция в JavaScript");
}
```

Сама по себе эта конструкция бесполезна. Анонимные функции почти всегда используются либо как аргументы для других функций (callback), либо присваиваются переменным.

### 2. Функциональное выражение (Function Expression)

Как показано в вашем примере, самый частый способ использования анонимной функции — это присвоить ее переменной. Этот паттерн называется **"Функциональное выражение"** (Function Expression).

```JavaScript
// 'display' — это переменная
// 'function() { ... }' — это анонимная функция (значение)
var display = function() {
  document.write("Анонимная функция в JavaScript" + "<br />");
};
```

Теперь функция (которая сама по себе безымянная) может быть вызвана через имя переменной, в которую ее "положили": `display();`

### 3. Ключевое отличие: Поднятие (Hoisting)

Это **самое важное** отличие от `Function Declaration` (когда мы писали `function cube(n) {}`).

- **Function Declaration (Обычная):** Полностью "поднимается" (hoist). Ее можно вызвать _до_ ее объявления в коде.
    
- **Function Expression (Как в примере):** **Не поднимается**.
    

Как показано в вашем коде, "поднимается" только сама **переменная** `var display` (и инициализируется значением `undefined`), но не _присваивание_ ей функции.

Поэтому этот код (из вашего примера) вызовет ошибку:

```JavaScript
// Ошибка:
// В этот момент 'display' существует, но его значение - 'undefined',
// а 'undefined' — это не функция, ее нельзя вызвать.
display(); 

var display = function() {
  // ...
};
```

### 4. Функции как "значения"

Этот пример также показывает, что в JavaScript функция — это просто "значение", как число или строка.

Переменной `display` сначала присвоили _одну_ функцию, а затем ее значение _перезаписали_, присвоив ей _другую_ анонимную функцию, которая работает иначе.



```JavaScript
// 1. Присвоили первую функцию
var display = function() {
  document.write("Анонимная функция..." + "<br />");
};
display(); // Вызывает первую функцию

// 2. Перезаписали переменную
display = function(number) {
  return number * number;
};

// 3. 'display' теперь ссылается на вторую функцию
document.write("Квадрат числа равен: " + display(5)); // Выведет 25
```
