## Функции в JavaScript

**Функция** — это именованный, многоразовый блок кода, предназначенный для выполнения определённой задачи. Функции являются фундаментальным строительным блоком любой JavaScript-программы.

Они позволяют "упаковать" логику, дать ей имя и затем вызывать (запускать) эту логику в любом месте программы столько раз, сколько потребуется.

### 1. Объявление функции (Function Declaration)

Это классический способ создания функции. Он использует ключевое слово `function`.

### 2. Ключевые компоненты (на основе вашего примера)

#### А. Функция без параметров и без `return`

Такие функции просто выполняют набор инструкций.

- **Пример из вашего файла:** `simpleFunction`
    
- **Объявление:**

 ```JavaScript
    function simpleFunction() {
      alert("Данная функция не принимает аргументов и не возвращает значение");
    }
    ```

- **Вызов:**

```JavaScript
 simpleFunction(); // Выполнит код внутри, покажет alert
 ```

- **Результат:** Эта функция выполняет действие, но ее "результат" как переменной равен `undefined`, так как в ней нет `return`.

#### Б. Функция с параметрами и `return`

Такие функции принимают входные данные (аргументы), обрабатывают их и возвращают результат.

- **Пример из вашего файла:** `cube`

- **Объявление:**

```JavaScript
    function cube(n) { // 'n' — это параметр
      var c = n * n * n;
      return c; // Возвращаем вычисленное значение
    }
    ```

- **Вызов:**
- 
 ```JavaScript
    // '3' и '5' — это аргументы, которые передаются в параметр 'n'
    alert("Куб числа " + 3 + " равен " + cube(3)); // cube(3) вернет 27
    alert("Куб числа " + 5 + " равен " + cube(5)); // cube(5) вернет 125
    ```
    

---

### 3. Важные понятия

- **Параметры** — это переменные-заполнители, перечисленные в скобках при _объявлении_ функции (как `n` в `cube(n)`). Они получают свои значения в момент вызова.
    
- **Аргументы** — это _реальные значения_, которые передаются в функцию при её _вызове_ (как `3` и `5`).
    
- **`return`** — это ключевое слово, которое:
    
    1. Немедленно **останавливает** выполнение функции.
        
    2. **Возвращает** указанное значение в то место, где функция была вызвана.
        
    3. Если `return` не указан (как в `simpleFunction`), функция автоматически возвращает `undefined`.
        
- **Вызов функции (Function Call)** — это "запуск" функции. Он выполняется путем написания имени функции с последующими круглыми скобками `()`, в которые передаются аргументы (если они нужны).
    

### 4. Поднятие (Hoisting) функций

Объявления функций (Function Declarations), созданные с помощью `function ...`, **полностью "поднимаются" (hoist)**.

Это означает, что JavaScript "знает" о них до начала выполнения кода. Вы можете безопасно _вызвать_ такую функцию в коде _до_ ее фактического объявления.

**Пример:**



```JavaScript
// Вызов работает, хотя функция объявлена ниже
sayHello(); // Выведет: "Привет!"

function sayHello() {
  console.log("Привет!");
}
```
