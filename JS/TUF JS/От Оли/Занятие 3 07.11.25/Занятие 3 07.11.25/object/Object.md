Объекты в JavaScript — это основная структура данных. Почти все в JavaScript является объектом или ведет себя как объект.

По своей сути, объект — это **неупорядоченная коллекция связанных данных**, которая хранится в виде пар **"ключ: значение"** (key: value pairs).

- **Ключ (Key):** Это строка (или Symbol), которая служит именем (идентификатором) для значения.
    
- **Значение (Value):** Может быть чем угодно: строкой, числом, булевым значением (`true`/`false`), массивом, другим объектом или даже функцией.
    

Когда значением является функция, ее называют **методом** объекта. В остальных случаях ключ и значение вместе называют **свойством** объекта.

---

## 1. Создание объекта (Литерал объекта)

Самый простой и распространенный способ создать объект — использовать **литеральную нотацию** (фигурные скобки `{}`).



```JavaScript
// Создаем пустой объект
let emptyObject = {};

// Создаем объект "user" с несколькими свойствами
let user = {
  name: "Alex",              // Свойство: ключ "name", значение "Alex" (строка)
  age: 30,                 // Свойство: ключ "age", значение 30 (число)
  isAdmin: true,           // Свойство: ключ "isAdmin", значение true (булево)
  "likes programming": true  // Ключ из нескольких слов (нужны кавычки)
};
```

## 2. Доступ к свойствам

Есть два способа получить доступ к значению свойства:

### A. Точечная нотация (Dot notation)

Это самый частый способ. Он используется, когда ключ является валидным именем переменной (без пробелов, дефисов, не начинается с цифры).


```JavaScript
console.log(user.name); // Выведет "Alex"
console.log(user.age);  // Выведет 30
```

### Б. Скобочная нотация (Bracket notation)

Этот способ более гибкий и требуется в двух случаях:

1. Когда ключ содержит пробелы, дефисы или другие спецсимволы.
    
2. Когда имя ключа хранится в переменной.
    


```JavaScript
// 1. Ключ с пробелом
console.log(user["likes programming"]); // Выведет true

// 2. Имя ключа в переменной
let key = "name";
console.log(user[key]); // Выведет "Alex" (то же самое, что user.name)
```

---

## 3. Изменение, добавление и удаление свойств

Объекты в JavaScript **мутабельны** (их можно изменять) после создания.


```JavaScript
let car = {
  color: "red"
};

// 1. Добавление нового свойства
car.brand = "Tesla";
console.log(car.brand); // Выведет "Tesla"

// 2. Изменение существующего свойства
car.color = "blue";
console.log(car.color); // Выведет "blue"

// 3. Удаление свойства
delete car.brand;
console.log(car.brand); // Выведет "undefined"
```

---

## 4. Методы объекта

Когда значением свойства является функция, это называется **методом**.


```JavaScript
let person = {
  name: "Alice",
  age: 25,

  // Это метод (традиционная запись)
  greet: function() {
    console.log("Hello!");
  },

  // Это метод (современная, более короткая запись)
  sayHi() {
    console.log("Hi!");
  }
};

// Вызов методов
person.greet(); // Выведет "Hello!"
person.sayHi(); // Выведет "Hi!"
```

### Ключевое слово "this"

Внутри метода, ключевое слово `this` ссылается на **сам объект**, которому принадлежит этот метод. Это позволяет методу получать доступ к другим свойствам _этого же_ объекта.



```JavaScript
let calculator = {
  a: 5,
  b: 10,

  sum() {
    // this.a ссылается на calculator.a
    // this.b ссылается на calculator.b
    return this.a + this.b;
  },

  multiply() {
    return this.a * this.b;
  }
};

console.log(calculator.sum());      // Выведет 15
console.log(calculator.multiply()); // Выведет 50
```
