Бинго! Именно `window.location`.

Внутри этого объекта хранится куча всего полезного про текущий адрес, но для твоего задания нас интересует конкретное свойство: **`window.location.hash`**.

Оно хранит именно ту часть, которая идет после решетки `#`.

- Если адрес `site.com/#about`, то `window.location.hash` вернет строку `"#about"`.
    

### Следующий шаг: "Слушаем" изменения

Теперь нам нужно научить JavaScript реагировать каждый раз, когда этот хвостик меняется. Для этого мы используем «слушателя событий» (event listener) на самом окне (`window`).

Событие называется `'hashchange'`.

Пусть для начала он просто выводит в консоль (console.log) текущий хэш, чтобы мы убедились, что всё работает.
