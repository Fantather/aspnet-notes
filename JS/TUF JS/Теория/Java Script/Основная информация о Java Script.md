*JavaScript* - это высокоуровневый интерпретируемый язык программирования, который используется для создания динамических и интерактивных приложений

Был создан для анимаций, для того что бы оживить HTML страницу, а теперь используется что бы принимать ответы от сервера

JavaScript является реализацией стандарта ECMAScript

JavaScript динамически типизованный язык, но он позволяет создавать переменные с неопределённым типом данных, undefined 

##### Потому что JavaScript умеет
•Валидировать данные форм форм
перед отправкой их на сервер
•Отправка запросов на сервер и загрузка
данных без полной перезагрузки страницы (AJAX)

• Вызывать функции программного интерфейса

• Получать доступ к файлам и другим системным ресурсам клиента и сервера
