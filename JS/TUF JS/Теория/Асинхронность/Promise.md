`Promise` -  Объект класса, который возвращает состояние Асинхронной операции и позволяется снаружи получить результат успешного выполнения (`then`) или результат неудачного выполнения(`catch`)

Асинхронные операции
- загрузка данных с сервера
- таймеры
- чтение файлов
- ожидание событий


`resolve` - возвращает результат
`regect` - исключение
