##### Объявление переменной
`let` - Современный способ объявления переменной, это создаёт *локальную* переменную
`var` - Устаревший способ объявления переменной, это создаёт *глобальную* переменную
Так же объявленные так переменные *всплывают* (Hoisting), то есть мы можем обратится к ним до того как мы объявили их в коде, они будут undefined

Это происходит потому что компилятор переносит их в начало области видимости, на пример в начало функции


##### Инициализация и присваивание переменной
```JS
let num = 1; // Число
num = "1"; // Строка
```
