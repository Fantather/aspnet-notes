GET - запит web сторінки
POST - передача данних на сервер (например отправить комент)
HEAD - запит заголовка сторінки
PUT - розміщення сорінки на сервері
DELETE - видалення сторінки з сервера
TRACE - трасування сторінки
OPTIONS - запрос підтримуванних методів НТТР для данного ресурсу
CONNECT - підключення до серверу через ргоху
