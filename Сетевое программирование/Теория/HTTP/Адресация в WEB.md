*Uniform Resource Identifier (URI)* - уникальная строка, которая глобально идентифицирует ресурс в сети 

Підвидом URI являеться Uniform Resource Locator (URL) - універсальний стандарт визначення розміщення ресурсів у мережі
универсальный стандарт, который определяет, как именно
