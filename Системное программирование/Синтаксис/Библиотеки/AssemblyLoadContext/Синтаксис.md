Если мы хотим использовать класс из библиотеки
![[Pasted image 20251009201559.png]]
`typ` - Перечисление классов внутри библиотеки
`cls` - `object`, который хранит внутри себя класс из библиотеки
`typ.GetMethod(Название метода).Invoke(Метод, параметры)` - вызываем метод внутри object, мы не можем просто распаковать класс, потому что компилятор не знает о нём
