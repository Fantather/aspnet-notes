Подготовка к использованию
```C#
CancellationTokenSourse cts = new CancellationTokenSourse();

public void MyMethod(CancellationToken ct)
{
	while(true)
	{
		ct.ThrowIfCancellationRequested();
		// Ещё какие-то действия
	}
}
```


Использование с Task
```C#
// Запустил метод в другом фоновом потоке
Task task = Task.Run(() => MyMethod(cts.CancellationToken), cts.CancellationToken);

cts.Cancel();
cts.Dispose();
cts = new CancellationTokenSourse();
```
`cts.Cancel()` - Метод внутри task выбросит исключение, когда дойдёт до строчки ct.ThrowIfCancellationRequested();
`cts.Dispose()` - Освобождаем ресурсы
