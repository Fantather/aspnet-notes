Используется для интесивных расчётов используя (multicore) многоядерность или многопроцессорность (multiprocess)
(Медианы, средние значения)

TPL - Task Parallel Library
Используется Threading.Task и DataParallelism, то есть вместо for и foreach есть циклы с параллелизмом (Parralel.For)

Каждый метод попытается запуститься на отдельном ядре, при возможности
Но мы не контролируем этот процесс и не можем это гарантировать

Данные необходимо синхронизировать, на пример через lock
Что бы этого избежать, данные делят на участки так, что бы разные участки кода не пересекались


Для работы с Parralel используют Concurrent виды коллекций
есть Parallel LINQ, вызываются через метод `.AsParallel`
