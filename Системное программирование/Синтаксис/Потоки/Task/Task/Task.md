*Task* - класс для простой отправки метода в [[Планировщик задач .NET|ThreadPool]], то есть Планировщику потоков .NET

#### Про исключения внутри Task
Каждый Task, после окончания его работы нужно проверять на исключения

Иначе, в .NET 4.0 и ниже, когда за Task с исключением внутри приходил [[GC|Сборщик мусора]], то он принудительно выбрасывал это исключение в свой поток FinalizerThread, что бы ты не пропустил это исключение, из-за чего всё приложение аварийно завершало свою работу 

С .NET 4.5 все незамеченные исключения внутри объекта Task исчезают вместе с убийством Task
Это привело к тому, что без проверки наш метод в потоке может постоянно выбрасывать исключения, но из-за того что они тихо "проглатываются" - мы не скоро узнаем, что что-то идёт не так
