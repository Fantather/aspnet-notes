**Callback-функция**
Наиболее предпочтительный подход
Метод, указанный в параметре `callback`, вызывается автоматически после завершения асинхронной операции

*Callback-функция* имеет строгую сигнатуру
`void AsyncCallback(IAsyncResult ar)`
Внутри нее необходимо вызвать `EndInvoke` для получения результата или обработки возможных исключений

**Использование дескриптора ожидания**
Блокировка потока до завершения с помощью `AsyncWaitHandle.WaitOne()` (может привести к взаимоблокировке в UI, избегайте этого)


**Опрос статуса (Polling)**
Цикл с проверкой `IsCompleted
```csharp
while (!asyncResult.IsCompleted) {
	Console.Write(".");
	Thread.Sleep(50);
}
```
