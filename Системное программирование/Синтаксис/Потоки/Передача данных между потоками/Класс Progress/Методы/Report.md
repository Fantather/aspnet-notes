Используется для сообщения об очередном изменении прогресса

Это единственный метод, который определяет интерфейс `IProgress<T>`
Его вызов просто запускает прописанный в UI-потоке метод и передаёт в него значение

Сигнатура:
```C#
void Report(T value)
```
Значение типа `T` передается в callback-функцию, которая была зарегистрирована при создании экземпляра класса `Progress<T>`
