- **Close** — освобождает все ресурсы, которые использовались связанным процессом

- **CloseMainWindow** — инициирует закрытие главного окна процесса

- **GetCurrentProcess** — статический метод; возвращает объект текущего процесса, в котором выполняется вызывающий поток

- **GetProcessById** — возвращает процесс по его PID (идентификатору)

- **GetProcesses** — возвращает массив объектов `Process` — список всех процессов, запущенных в операционной системе

- **GetProcessesByName** — возвращает процессы по их имени

- **Kill** — немедленно завершает процесс

- **OnExited** — инициирует событие `Exited`

- **Refresh** — обновляет сведения об объекте `Process`, отменяя накопленные изменения с момента его создания

- **Start** — запускает процесс

- **WaitForExit** — заставляет вызывающий поток ждать завершения процесса заданное время
