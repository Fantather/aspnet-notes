- **CreateInstance** — создаёт экземпляр указанного типа из этой сборки по имени.

- **GetAssembly** — возвращает сборку, в которой объявлен указанный класс.

- **GetCallingAssembly** — возвращает сборку, из которой вызван текущий метод.

- **GetCustomAttributes** — возвращает атрибуты, установленные для этой сборки.

- **GetExecutingAssembly** — возвращает сборку, содержащую выполняемый в данный момент код.

- **GetExportedTypes** — типы, объявленные в сборке и доступные извне.

- **GetLoadedModules** — массив всех модулей, загруженных сборкой.

- **GetModule** — конкретный модуль по имени.

- **GetModules** — массив всех модулей текущей сборки.

- **GetName** — объект `AssemblyName`, описывающий текущую сборку.

- **GetReferencedAssemblies** — список сборок, на которые ссылается эта сборка.

- **GetType** — объект `Type`, описывающий конкретный тип данных по имени.

- **GetTypes** — массив всех типов данных, описанных в сборке.

- **IsDefined** — проверяет, применён ли к сборке указанный атрибут.

- **Load** — загружает сборку по полному имени или по объекту `AssemblyName`.

- **LoadFile** — загружает сборку из указанного файла.

- **LoadFrom** — загружает сборку по имени или пути к файлу.

- **LoadModule** — загружает модуль в текущую сборку.

- **ReflectionOnlyLoad** — загружает сборку в режиме только для отражения (код не может выполняться).
