- **EntryPoint** — точка входа сборки (обычно метод `Main`).

- **Evidence** — объект `Evidence`, описывающий разрешения политики безопасности, предоставленные сборке.

- **FullName** — полное имя сборки.

- **GlobalAssemblyCache** — логическое значение: загружена ли сборка из глобального кэша (GAC).

- **ImageRuntimeVersion** — версия среды выполнения .NET, для которой собрана сборка.

- **Location** — путь к файлу сборки.

- **ManifestModule** — объект `Module`, содержащий манифест текущей сборки.

- **ReflectionOnly** — `true`, если сборка загружена только для отражения (просмотра), без возможности выполнения.
