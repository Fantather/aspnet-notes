- **Invoke** — выполняет определённый метод, объявленный в этом модуле

- **GetCustomAttributes** — массив всех атрибутов модуля

- **GetField** — объект `FieldInfo`, описывающий поле класса, объявленного в модуле

- **GetFields** — массив `FieldInfo` для всех полей заданного класса (или `null`, если класс не найден)

- **GetMethod** — объект `MethodInfo` для метода с указанным именем (или `null`, если такого метода нет)

- **GetMethods** — массив `MethodInfo` для всех методов модуля

- **GetType** — объект `Type` для типа данных, объявленного в модуле

- **GetTypes** — массив `Type` для всех типов данных модуля

- **IsDefined** — проверяет, задан ли для модуля указанный атрибут

- **IsResource** — показывает, является ли модуль ресурсным (только для хранения ресурсов)
