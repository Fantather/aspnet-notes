- **Assembly** — объект `Assembly`, которому принадлежит модуль.

- **FullyQualifiedName** — полное имя модуля, включая путь.

- **ModuleHandle** — дескриптор модуля (может использоваться при вызове Win32 API).

- **ModuleVersionId** — уникальный идентификатор версии модуля.

- **Name** — имя модуля
