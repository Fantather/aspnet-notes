**Установление приоритета**
![[Pasted image 20250923190238.png]]
```C#
Process proc = new Process();
proc.PriorityClass = ProcessPriorityClass.High;
```

`PriorityClass` - Enum, который хранит предустановленные значения для `BasePriority`
`RealTime` - Приоритет для Системных процессов, нашим процессам такое присвоить нельзя
`Normal` - Используется по умолчанию
`Idle` - Для фоновых процессов, которые могут позволить себе ожидание


**Имеет ли процесс окно**
`MainWindiwHandle` - bool
