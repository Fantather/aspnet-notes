Создание класса, который будет хранить аргументы
```C#
ProcessStartInfo startInfo = new ProcessStartInfo("notepad.exe");
```

Запуск с одним аргументом
```C#
startInfo.Arguments = "M.txt";
startInfo.WorkingDirectory = @"F:\";

Process.Start(startInfo);
```
`startInfo.Arguments` - Файл который запустит процесс
`startInfo.WorkingDirectory` - Директория в которой нужно искать файл, который будет запущен вместе с процессом

`Process.Start(startInfo);` - Запускаем процесс с аргументами


```C#
Process proc=new Process();
proc.StartInfo.FileName = "notepad.exe";
proc.StartInfo.Arguments= "M.txt";
proc.StartInfo.WorkingDirectory= @"F:\";

proc.Start();
```
