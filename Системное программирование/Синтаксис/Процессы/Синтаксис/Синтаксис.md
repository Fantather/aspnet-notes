Создание процесса

В *реестре Windows* хранятся пути к исполняемым файлам программ зарегистрированным в реестре под определенным именем *notepad.exe* или указываем путь к файлу

```C#
Process proc = new Process();
proc.StartInfo.FileName = "notepad.exe";

Console.WriteLine(proc.BasePriority);

proc.Start();
proc.PriorityClass = ProcessPriorityClass.RealTime;


Console.WriteLine(proc.BasePriority);

//      BasePriority      PriorityClass
//           4                Idle
//           8               Normal
//           13               High
//           24              RealTime
```
